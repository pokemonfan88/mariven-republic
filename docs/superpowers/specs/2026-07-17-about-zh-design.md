# “关于马里文”中文页面设计

## 目标

在 `马里文政府官网/马里文政府门户-网站文件/` 中新增 `about-zh.html`，为现有 `about.html` 提供完整的简体中文版本。

## 页面结构与样式

- 保留英文页面的全部区块、顺序、表格、图片和 CSS 类名。
- 继续引用 `assets/css/site.css`、`assets/flag.svg` 和 `assets/seal.png`，不新增样式或图片资源。
- 文档语言设为 `zh-CN`，编码保持 UTF-8，并翻译页面标题和描述元信息。

## 翻译范围

- 完整翻译顶部日期、语言栏、气象警报、站点标题、导航、面包屑、正文各章节、图片替代文字和页脚。
- 数字、年份、统计值、货币代码、电话号码、网址和专有缩写保持原有事实含义。
- 地名、人名、机构名和宪政术语与现有 `zh.html`、`government-zh.html` 保持一致。
- 马里文语名称、格言和国歌名在适合的位置保留原文，并提供中文释义。

## 链接规则

- 本页语言栏的 English 指向 `about.html`，中文链接指向 `about-zh.html` 并显示为当前语言。
- 主导航指向中文页面：`zh.html`、`government-zh.html`、`zh.html#services`、`zh.html#news`、`about-zh.html` 和 `zh.html#contact`。
- 正文中的政府领导人链接指向 `government-zh.html`。
- 原页面中尚未配置目标的占位链接继续保留为 `#`。

## 验证

- 检查 HTML 标签结构、`lang`、UTF-8 元信息和资源路径。
- 检查页面是否残留应翻译的英文界面文案或出现乱码。
- 检查中英文切换、中文主导航和正文内部链接。
- 在浏览器中渲染页面，确认桌面布局、中文换行、表格和页脚显示正常。

## 非目标

- 不修改英文版 `about.html`。
- 不重写网站样式，不调整页面布局，不添加新功能。
- 不补充或更改原英文页面所载事实数据。
