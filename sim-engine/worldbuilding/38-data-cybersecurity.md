# 数据保护与网络安全

## 数据保护法

### 《数据保护法》（Data Protection Act, 2021年）

| 项目 | 详情 |
|------|------|
| 全名 | 《2021年个人数据保护法》 |
| 生效 | 2022年1月1日 |
| 参照 | 以欧盟 GDPR 为蓝本，但更简化 |
| 监管机构 | 马里文数据保护专员办公室（Office of the Data Protection Commissioner / ODPC） |
| 数据保护专员 | 萨拉·莫顿（Sarah Morton），2022年任命 |

### 核心规定

| 原则 | 内容 |
|------|------|
| **同意** | 收集和处理个人数据须获得明确的知情同意 |
| **目的限制** | 数据只能用于收集时声明的目的 |
| **数据最小化** | 只收集完成目的所需的最少数据 |
| **存储限制** | 数据保留时间不超过必要期限 |
| **数据主体权利** | 访问权、更正权、删除权（"被遗忘权"）、数据可携权 |
| **数据泄露通知** | 重大数据泄露须在 72 小时内通知 ODPC 和受影响主体 |
| **跨境数据传输** | 只允许传输至"充分保护水平"的国家（目前认可：欧盟、英国、澳大利亚、新西兰） |

### 处罚

- 违规罚款：最高 $250,000 MVL 或全球年营收 2%（取较高者）
- 刑事处罚：故意或重大过失导致的数据泄露可判监 2 年
- **2023年首次执法**：马卡迪港一家连锁超市因未经同意向第三方出售客户购物数据被罚款 $35,000 MVL

### 争议

- **中小企业合规成本**：小型企业抱怨 GDPR 式的要求过于复杂——"我们只有 5 个员工但还是要写 30 页的隐私政策"
- **中国数据传输**：2025年中马自贸协定签署后，涉及中国 IT 公司的数据传输是否符合"充分保护"标准成为争议焦点
- **政府自身豁免**：国家安全和执法部门可以豁免于部分规定——公民自由团体批评这个豁免太宽泛

## 网络安全

### 马里文计算机应急响应小组（Mariven CERT）

| 项目 | 详情 |
|------|------|
| 设立 | 2016年 |
| 隶属 | 数字基础设施部 |
| 人员 | 18 人（含网络安全分析师、取證专家） |
| 职能 | 网络攻击监测和响应、漏洞预警、安全事件调查 |
| 年度预算 | $180万 MVL |

### 网络安全态势

| 指标 | 数值/状况 |
|------|-----------|
| 2025年报告的网络事件 | 1,280 起（大部分为钓鱼邮件和勒索软件） |
| 重大事件 | 2024年国家统计局网站遭 DDoS 攻击（持续 6 小时，未造成数据泄露） |
| 2023年马卡迪港电力公司勒索软件攻击 | 支付 $120,000 MVL 赎金（未公开承认）后被内部人士爆料——引发全国网络安全辩论 |
| 网络犯罪报案率 | 低（估计仅 20% 的诈骗案件被举报） |

### 主要威胁

| 威胁类型 | 详情 |
|------|------|
| **网络钓鱼** | 最常见——伪装成银行或税务局邮件骗取密码 |
| **勒索软件** | 2022-2025年间有 8 起企业勒索案件被公开 |
| **社交媒体诈骗** | Facebook 和 WhatsApp 上的欺诈信息泛滥 |
| **APT 攻击**（国家级） | 2024年 CERT 发现一起针对外交部电子邮件系统的"中等复杂度"的网络间谍活动——未指认来源国 |

## 《网络安全法》（Cybercrime Act, 2019年）

| 条款 | 内容 |
|------|------|
| 未经授权访问计算机系统 | 最高监禁 2 年 |
| 数据干扰 | 最高监禁 5 年 |
| 网络诈骗 | 根据诈骗金额处罚，最高 10 年 |
| 儿童色情 | 最高 15 年（与国际刑警组织合作） |
| 网络骚扰（Cyberbullying） | 最高监禁 1 年 + 罚款 $5,000 MVL |

### 《网络安全法》修正争议（2024年修正案）

2024年通过的修正案新增了"传播虚假信息"条款——授权政府在"公共紧急状态"期间要求平台删除被判定为"不实"的内容。批评者认为这为政治审查开了后门。2025年一名博主因在 Facebook 上发布"气旋马莱卡政府救灾款被贪污"的帖子而被调查（最终未被起诉），加剧了言论自由的担忧。

## 网络安全挑战

| 挑战 | 说明 |
|------|------|
| **人才极度短缺** | 全国仅有约 30 名持证网络安全专业人员。优秀人才几乎全部被澳新公司远程招聘挖走 |
| **关键基础设施脆弱** | 电力系统、港口控制系统、电信网络均缺乏足够的网络安全防护 |
| **意识不足** | 中小企业几乎不投资网络安全——"我们太小，黑客不会盯上我们"（实际情况正好相反） |
| **国际合作依赖** | 重大攻击的溯源能力严重依赖澳大利亚和新西兰的网络安全机构 |
