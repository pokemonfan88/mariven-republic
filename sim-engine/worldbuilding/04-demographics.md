## 民族构成

马里文是一个多民族社会，四大族群和谐共处——

### 1. 马里文原住民（Marivenese Indigenous / Kana Motu）

| 指标 | 数值 |
|------|------|
| 占人口比例 | **58%**（约69.6万） |
| 起源 | 南岛语族，约公元前1500年抵达 |
| 主要分布 | 全境，以蒂莫岛和鲁瓦岛比例最高（>80%） |
| 传统社会结构 | 母系氏族（Matavuvale），土地由氏族共有 |
| 传统经济 | 自给农业、渔业、卡瓦种植 |
| 社会地位 | 宪法承认原住民土地权和传统酋长体系 |

**文化特征**：卡瓦仪式（Kava Ceremony）、Meke 传统舞蹈、Tapa 树皮布工艺。原住民社会以社区互助（Solesolevaki）为核心理念。

### 2. 马里文-印度裔（Marivenese Indians / Kana Initia）

| 指标 | 数值 |
|------|------|
| 占人口比例 | **28%**（约33.6万） |
| 起源 | 1878-1916年英属印度契约劳工后代 |
| 主要分布 | 主岛西部（马卡迪港周边）、卡托拉市 |
| 使用语言 | 马里文-印地语方言（Fiji Hindi 的变体）、英语、马里文语 |
| 宗教信仰 | 印度教（绝大多数）、伊斯兰教（约15%的印度裔） |
| 经济角色 | 甘蔗种植、零售业、运输业、专业服务业 |

**文化特征**：排灯节（Diwali）为国家公共假日、印度裔饮食深刻影响了马里文国菜（咖喱鱼、Roti 卷饼）、宝莱坞电影广受欢迎。

### 3. 欧洲裔（European Marivenese / Kana Europa）

| 指标 | 数值 |
|------|------|
| 占人口比例 | **6%**（约7.2万） |
| 起源 | 英国殖民者、传教士、商人后代 |
| 主要分布 | 卡托拉市、佩拉岛 |
| 使用语言 | 英语（母语）、部分通晓马里文语 |

**历史角色**：控制商业和种植园至1970年。独立后多数留在马里文，融入社会。现多从事金融、法律、教育等行业。

### 4. 华裔（Chinese Marivenese / Kana Saina）

| 指标 | 数值 |
|------|------|
| 占人口比例 | **4%**（约4.8万） |
| 起源 | 20世纪初自广东、福建移民 |
| 主要分布 | 卡托拉市唐人街区、佩拉岛旅游区 |
| 使用语言 | 粤语、客家话、英语、马里文语 |

**经济角色**：零售业、餐饮业、进出口贸易。卡托拉市的「中华商会」成立于1925年，是该国最老的商会之一。年春节为地方性节日，卡托拉市有舞狮游行。

### 5. 其他太平洋岛民（Pacific Islanders / Kana Pasifika）

| 指标 | 数值 |
|------|------|
| 占人口比例 | **4%**（约4.8万） |
| 来源 | 虚构邻国、汤加、萨摩亚、所罗门群岛等移民 |
| 主要分布 | 卡托拉市、旅游区 |

## 宗教

马里文的宗教格局是殖民史和移民史的直接映射——

### 主要信仰分布

| 宗教 | 人口比例 | 人数 |
|------|----------|------|
| 基督教 | 64% | 768,000 |
| －卫理公会（Methodist） | 36% | 432,000 |
| －天主教（Catholic） | 18% | 216,000 |
| －其他新教 | 10% | 120,000 |
| 印度教 | 22% | 264,000 |
| 伊斯兰教（逊尼派） | 5% | 60,000 |
| 传统信仰 | 6% | 72,000 |
| 无宗教/其他 | 3% | 36,000 |

### 宪法与宗教自由

- 《宪法》第4条：马里文是世俗国家，政教分离。
- 第21条：保障宗教信仰自由，禁止宗教歧视。
- 星期日为法定休息日（基督教的殖民遗产）。
- 印度教排灯节和伊斯兰教开斋节均为国家公共假日。

### 宗教场所

- 卡托拉圣三一大教堂（Holy Trinity Cathedral）— 卫理公会，1870年建
- 马卡迪港印度教寺庙群 — 全岛最大，4座寺庙组成
- 卡托拉中央清真寺 — 2010年新建，可容纳1500人
- 塔普火山传统圣地 — 原住民举行祈福仪式

## 社会结构

### 城乡差异

- 城市（卡托拉市、马卡迪港）：53.6%的人口，服务业和工业就业为主
- 农村：以村庄（Koro）为单位，自给农业+经济作物
- 外岛（蒂莫、鲁瓦）：生活节奏缓慢，基础设施落后于主岛

### 性别

- 女性劳动参与率：42%（2025年）
- 议会女性议员比例：19%（15/78席）
- 女性受教育程度：与男性基本持平
- 土地继承：原住民传统为母系继承，但现代法律框架为性别中立

### 侨民 (Diaspora)

- 海外马里文人估计：约15万人
- 主要国家：澳大利亚（5万）、新西兰（4万）、加拿大（2万）、英国（1.5万）、美国（1.5万）
- 侨汇收入：约2.1亿美元/年（占GDP 2.9%）
- 侨民在圣诞节和独立日期间大规模返乡
