# 酒类与烟草管制

## 酒类

### 饮酒概况

| 指标 | 数值 |
|------|------|
| 年人均纯酒精消费（15+） | 5.2 升 |
| 男性饮酒者比例 | 约 58%（过去12个月内） |
| 女性饮酒者比例 | 约 22% |
| 酗酒率 | 约 18%（男性 28%、女性 8%） |
| 酒精性肝病 | 第五大死因 |

**饮酒模式**：马里文属于典型的"狂饮文化"（binge-drinking culture）——大多数饮酒者不是每天都喝，而是一次喝很多。"不发工资时不喝，发了工资喝到倒"。

### 合法饮酒年龄

| 类型 | 年龄 |
|------|------|
| 购买/饮用任何酒类 | **18 岁** |
| 在酒吧/餐厅饮用 | 18 岁（执行较严——酒吧查 ID 相对普遍） |
| 超市购买外带 | 18 岁（执行松——青少年常能买到啤酒） |

### 售酒管制

| 规定 | 详情 |
|------|------|
| 超市售酒时间 | 周一至周六 9:00-21:00 |
| 酒吧/餐厅售酒时间 | 可至凌晨 1:00（需持酒吧牌照） |
| **周日禁酒令** | 超市禁止在周日售卖外带酒精——基督教殖民遗产。酒吧和餐厅除外。这一"蓝色法律"每几年就引发一次废除讨论，但每次都被教会成功抵制。 |
| 特殊活动临时售酒许可 | 社区节日、婚礼可申请临时许可——审批视当地酋长意见而定 |

### 酒精广告限制

| 限制 | 详情 |
|------|------|
| 电视/广播 | 可在 20:00-05:00 时段播出 |
| 户外广告 | 允许，但距离学校 200m 以内禁止 |
| 体育赞助 | 允许——啤酒品牌是橄榄球的主要赞助商。"马里文勇士"队的球衣上印着"Katora Gold Beer" |
| 社交媒体 | 无具体限制（灰色地带——网红发一张举杯照不需要任何声明） |

### 酒驾

| 标准 | 详情 |
|------|------|
| 普通驾驶 | 血液酒精浓度 0.05%（2021年从 0.08% 下调） |
| 新手/职业司机 | 0.00%（零容忍） |
| 惩罚（首次） | 罚款 $800-2,500 MVL + 吊销驾照 6 个月 |
| 拒绝吹气测试 | 视为自动有罪，惩罚等同于酒驾标准 |
| 酒驾致死 | 构成"危险驾驶致人死亡"——最高监禁 10 年 |

**年度酒驾执法**（2025年）：进行了约 8,500 次呼气测试，发现了 1,020 例超标——酒驾率约 12%。周五和周六晚上是酒驾最高发时段。

## 烟草

### 吸烟概况

| 指标 | 数值 |
|------|------|
| 成人吸烟率 | 约 22%（男性 35%、女性 9%） |
| 年烟草消费 | 约 3.5亿支香烟（主要是进口品牌） |
| 青少年吸烟率（13-15岁） | 约 9%（男 12%、女 6%） |
| 吸烟相关死亡 | 估计每年约 380 人 |

**趋势**：吸烟率从 2010年的 31% 稳步下降至 2025年的 22%——烟草税连续五年大幅上调是最主要的驱动力。

### 烟草管制

| 规定 | 详情 |
|------|------|
| 合法吸烟年龄 | **18 岁**（2019年从 16 岁上调） |
| 烟草消费税 | $12 MVL/包（20支）——占总零售价约 65% |
| 一包万宝路价格 | $18.00 MVL（2019年 $8.50 → 每两年涨一次烟税） |
| 公共场所禁烟 | 《无烟环境法》（2014年）：所有室内公共场所、公共交通工具、工作场所全面禁烟 |
| 餐厅/酒吧 | 全面禁烟（含露台——2019年扩展） |
| 烟草广告 | 全面禁止（含促销和赞助） |
| 健康警示 | 烟盒正面 50% 面积、背面 75% 面积须印健康警示图文 |
| **平装法（Plain Packaging）** | 2022年通过，2024年实施——成为继澳大利亚、新西兰之后太平洋第三个强制平装的国家 |

### 电子烟与加热烟草

| 类型 | 法律状态 |
|------|----------|
| 含尼古丁电子烟 | **合法**（自 2020年）——但销售须获许可 |
| 向未成年人销售 | 非法（与烟草相同） |
| 公共场所使用 | 与烟草相同的禁烟规定 |
| 争议 | 2023年以来电子烟在青少年中"爆发式流行"——学校老师说"厕所里全是水果味的蒸汽"。卫生部正在审查是否收紧电子烟管制。 |

### 传统吸烟

- 烟草不是马里文的传统作物——吸烟习惯全系殖民时期引入
- 某些原住民长者会吸食一种本土草本（Kura Vola）的叶子——与传统仪式有关，但也逐渐被机制香烟取代

### 戒烟支持

- 戒烟热线（免费 0800-QUIT-MV）——年约 1,200 人次拨打
- 医生可开处尼古丁替代疗法（NRT）贴片——NHIS 每年补贴 2 个疗程
- 但农村地区戒烟资源几近空白——"想戒烟全靠意志力，因为最近的诊所离你 3 小时船程"
