# 社会福利

## 概述

马里文的福利体系起步较晚，以「安全网」而非「全民福利」为目标。核心支柱包括国家养老金、贫困家庭现金补助、灾害救济和免费教育。覆盖面有限，尤其在非正式经济部门（约占劳动力 35%）。

---

## 养老金体系

### 国家养老金计划（National Pension Scheme / NPS）

| 项目 | 详情 |
|------|------|
| 设立 | 1975年 |
| 类型 | 强制储蓄 + 政府最低保障 |
| 缴费率 | 工资的 8%（雇员）+ 8%（雇主） |
| 领取年龄 | 60岁 |
| 覆盖范围 | 正式部门雇员（约 28万人） |

### 老年人最低生活保障（Senior Citizens Allowance）

| 项目 | 详情 |
|------|------|
| 设立 | 2005年 |
| 对象 | 65岁以上、无养老金、月收入低于 $300 MVL |
| 金额 | $120 MVL/月 |
| 受益人数 | 约 18,000 人 |
| 年支出 | $2,600万 MVL |

**争议**：金额仅够基本食物，远低于贫困线。2024年反对党（DPA）提出将金额翻倍至 $240 MVL，政府以"财政不可持续"为由搁置。

## 家庭与儿童福利

| 项目 | 详情 |
|------|------|
| **儿童津贴**（Child Benefit） | 每个 0-18 岁子女 $45 MVL/月，仅限家庭月收入低于 $800 |
| **单亲家庭补助** | 额外 $80 MVL/月 |
| **产假** | 14 周带薪（雇主支付前8周 + NHIS 后6周） |
| **陪产假** | 3 天带薪 |
| **免费学校午餐** | 覆盖约 42% 农村小学（2018年起） |

## 失业与伤残保障

### 失业救济

- **无全国性失业保险制度**。
- 被解雇员工可获得一次性遣散费：每工作年 × 1 周工资（法定最低）
- 2024年议会曾讨论设立失业保险基金，但因企业反对仍未通过
- 实际失业保障主要依赖：家庭支持网络（Solesolevaki）+ 侨汇

### 伤残保障

| 项目 | 详情 |
|------|------|
| **工伤赔偿**（Workers Compensation） | 雇主强制保险，覆盖工作相关伤害和职业病 |
| **残疾福利**（Disability Allowance） | $150 MVL/月，需通过医学鉴定（约 4,200 人领取） |

## 贫困与社会救助

### 贫困数据

| 指标 | 数值 |
|------|------|
| 贫困线 | $1,850 MVL/人/年（国家贫困线） |
| 贫困率 | 14.8%（约 17.8万人，2025年） |
| 极端贫困率 | 2.2%（约 2.6万人，日生活费 < $2.15 USD） |
| 基尼系数 | 0.38（中等不平等） |

### 家庭补助计划（Family Assistance Program / FAP）

- 设立：2010年
- 对象：贫困线以下家庭
- 金额：$80-180 MVL/月（按家庭规模）
- 受益：约 15,500 户（约 7.2万人）
- 条件：学龄儿童须在校就读（Conditional Cash Transfer）
- 年支出：$2,800万 MVL

### 灾害救济

热带气旋后可启动紧急救济：
- 临时住所（学校/教堂改为避难所）
- 现金补助 $500 MVL/户（一次性）
- 食品包配送（军队参与）
- 2023年「气旋马莱卡」用于灾害救济的支出达 $1,200万 MVL

## 住房福利

- **无全国性公共住房计划**
- **首次购房者优惠**：免征印花税（房价 < $150,000 MVL）
- **公务员住房贷款**：优惠利率 3.5%（市场利率约 6.2%）
- **非正式定居点**：卡托拉市郊区有约 8 个非正式定居点（squatter settlements），约 3.2 万人居住。政府 2024 年启动「正规化计划」，缓慢推进。

---

## 社会服务的资金来源

| 来源 | 占比 |
|------|------|
| 政府一般预算 | 68% |
| 专项薪资税（NHIS等） | 22% |
| 国际援助 | 8% (澳大利亚、新西兰、联合国) |
| 慈善/教会 | 2% |

**社会支出占政府总支出**：约 28%（其中教育 14%、卫生 8%、社会福利 6%）

---

## 当前议题

1. **非正式部门覆盖**：约 35% 劳动力在非正式部门，完全不在任何社会保障网内
2. **老龄化**：65岁以上人口占比将从当前的 5.8% 升至 2040年的 12%，养老金体系承压
3. **侨汇依赖**：社会福利体系的结构性缺陷被侨汇（$2.1亿/年）填补——本质上"新西兰纳税人在补贴马里文老人"
