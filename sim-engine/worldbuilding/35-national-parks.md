# 国家公园与自然保护区

## 概述

马里文拥有 14,820 km² 的陆地面积和 1,290,000 km² 的专属经济区。虽然国土面积有限，但生态系统多样性极高——从火山雨林到珊瑚礁到红树林湿地。目前受保护区域占国土面积约 18%，海洋保护区占 EEZ 的 4.2%。政府承诺到 2030 年将海洋保护区覆盖率提升至 30%（"30x30" 倡议）。

## 国家公园体系

### 1. 塔普山国家公园（Tapu Mountain National Park）

| 项目 | 详情 |
|------|------|
| 设立 | 1985年（全国第一个国家公园） |
| 面积 | 382 km² |
| 地点 | 马卡迪岛中部高原 |
| 海拔范围 | 280m — 1,324m（塔普山顶） |
| 游客中心 | 2018年建成，位于公园南入口（塔普瓦莱方向） |

**核心景观**：
- 塔普火山口湖（Lake Tapu）：直径 2.8 km，深 82m，翠绿色水体——UNESCO 世界遗产核心区
- 塔普山主峰：攀登路线约 14 km（往返 7-9 小时），登顶可远眺四岛
- 岩画谷（Petroglyph Valley）：约 50 处公元 800-1200 年的火山岩刻画，需向导陪同

**生态**：
- 植被垂直带：低地雨林（<500m）→ 山地云雾林（500-1000m）→ 高海拔蕨类灌丛（>1000m）
- 特有物种：塔普湖盲虾（仅存在于火山口湖洞穴）、马里文飞狐

**年游客**：约 22,000 人（徒步者和自然爱好者）
**门票**：$8 MVL（成人）/$3 MVL（儿童）

### 2. 鲁瓦雨林保护区（Ruwa Rainforest Reserve）

| 项目 | 详情 |
|------|------|
| 设立 | 1998年 |
| 面积 | 260 km²（占鲁瓦岛 44%） |
| 地点 | 鲁瓦岛中央（岛屿北部为矿区，南部为村庄，中间为保护区） |
| 保护级别 | IUCN 第 II 类（国家公园级别） |
| 世界遗产状态 | 预备名单（自然遗产，2024年提交） |

**生态价值**：
- 马里文现存最大面积的未受干扰原始雨林
- 已知植物种类：超过 450 种维管植物，其中 28 种为马里文特有
- **国鸟唯一栖息地**：马里文翠鸟（Sikalé，约 600-800 只）仅存在于这片保护区
- 哺乳动物：马里文飞狐（翼展 1.2m）、5 种特有蝙蝠

**访问限制**：严格限制游客数量（每日上限 50 人，必须持许可并由指定向导陪同），以保护国鸟繁殖区。

**与矿业的关系**：
保护区的北部边界紧邻瓦图库拉金矿。矿区的爆破震动和酸性排水长期威胁保护区生态。2018年一项环境影响评估确认矿区排水导致下游河流重金属超标，此后矿业公司被要求安装水处理设施。2023年矿难后，环保团体呼吁将保护区边界外移 2公里，仍在争议中。

### 3. 科罗海海洋保护区（Koro Sea Marine Protected Area）

| 项目 | 详情 |
|------|------|
| 设立 | 2006年 |
| 面积 | 2,850 km²（海域） |
| 地点 | 马卡迪岛东岸科罗海 |
| 类型 | 多层次：禁渔核心区（15%）、有限捕捞区（40%）、传统渔区（45%） |

**生态**：
- 珊瑚覆盖率：约 68%（2024年调查）——在全球珊瑚退化的背景下，这已经是"幸存者乐园"
- 海龟筑巢地：绿海龟和玳瑁在东岸海滩筑巢
- 座头鲸迁徙通道：每年 7-10月可在科罗海观测到座头鲸

**管理**：由马里文海洋研究所和当地村庄共同管理。禁渔区的渔获量反而比开放区高出 40%（溢出效应），成为本地渔业社区支持保护的最有力证据。

### 4. 蓝湖海洋保护区（Blue Lagoon MPA）

| 项目 | 详情 |
|------|------|
| 设立 | 2010年 |
| 面积 | 480 km² |
| 地点 | 佩拉岛环礁潟湖 |
| 游客 | 年约 15万人次（浮潜和潜水，是参观量最高的保护区） |

**核心景观**：彩虹礁——珊瑚种类超过 200 种，是佩拉岛的头号旅游资产。"每潜一次 $220 MVL"的商业价值直接推动了保护——度假村自身成为最强有力的保护倡导者。

**2024年珊瑚白化事件**：全球海洋热浪导致彩虹礁约 22% 的珊瑚白化。目前处于恢复监测中。2024年马里文海洋研究所在彩虹礁启动了"耐热珊瑚辅助迁移"项目。

### 5. 蒂莫岛红树林保护区（Timo Mangrove Reserve）

| 项目 | 详情 |
|------|------|
| 设立 | 2012年 |
| 面积 | 58 km²（红树林） |
| 地点 | 蒂莫岛北岸和东岸 |
| 红树林物种 | 7 种（包括红茄苳、白骨壤） |

**功能**：
- 天然风暴屏障：2023年气旋马莱卡期间，红树林消减了约 40% 的风暴潮能量
- 鱼类育幼场：蒂莫岛 60% 的商业鱼类在红树林中度过幼年期
- 碳汇：储碳量估计约每公顷 850 吨 CO₂——是热带雨林的 3-5 倍

## 保护区管理体系

| 机构 | 职能 |
|------|------|
| **马里文国家公园管理局**（MNPA） | 陆地保护区管理 |
| **海洋保护区管理局**（MPA Authority） | 海洋保护区管理 |
| **村庄共管委员会** | 社区参与（每个保护区至少 1 个） |

**资金**：保护区年度预算约 $820万 MVL——严重不足。约 40% 来自旅游收入（门票、潜水许可费），35% 来自政府拨款，25% 来自国际援助（澳大利亚、全球环境基金）。
