# 赌博与彩票

## 概述

马里文的赌博文化复杂——传统上酋长阶层视赌博为道德败坏，这一禁忌部分被现代法律继承。全国仅有一座合法赌场（且只对外国游客开放），没有国家彩票。但地下非法赌博普遍存在，在线赌博的灰色地带也在扩大。教会是反对赌博合法化最强大的力量。

---

## 法律框架

| 类型 | 法律状态 | 详情 |
|------|----------|------|
| **赌场** | **合法但有限制** | 仅 1 家（佩拉岛赌场），且仅向外国游客开放——马里文公民被禁止进入 |
| **体育博彩** | **非法** | 任何形式的体育博彩在全国范围内非法 |
| **老虎机/电子游戏机** | **非法** | 除了佩拉岛赌场内的老虎机（仅限游客） |
| **国家彩票** | **不存在** | 多次提案，每次均被议会否决 |
| **在线赌博** | **灰色地带** | 无具体法律——理论上公民可通过 VPN 参与海外赌博网站；政府既不许可也不主动追查 |
| **宾果/抽奖** | **合法（慈善豁免）** | 教会和社区组织可申请一次性慈善博彩许可——Bingo 之夜是教堂地下室最受欢迎的筹款方式 |
| **动物斗赛**（斗鸡） | **名义非法，实质默认** | 法律上非法，但农村地区仍广泛存在——警察基本不管。在丰收节期间斗鸡几乎是半公开的活动。 |

## 佩拉岛赌场（Pela Casino）

| 项目 | 详情 |
|------|------|
| 开业 | 2012年（附属于蓝湖度假村） |
| 运营方 | 蓝湖度假集团（澳大利亚 60% + 本地 40% 合资） |
| 设施 | 8 张赌桌（百家乐、二十一点、轮盘）+ 45 台老虎机 |
| 游客限制 | 进场须出示外国护照——马里文公民即使持双国籍也以马里文公民身份计——不得进入 |
| 年游客人流 | 约 25,000 人次（占佩拉岛游客的 3%） |
| 年赌场收入 | 约 $1,800万 MVL |
| 赌场税 | 博彩总收入的 15%——约 $270万 MVL/年进入国库 |

### 公民禁入令的由来

- 独立后初期（1970年代），三个酋长大会的领袖联名请求政府全面禁止赌博——"赌场是英国的坏事，现在英国人走了，坏事也该走"
- 1998年首次提案开设赌场以吸引高端旅游——教会和酋长联合抗议，提案撤回
- 2008年妥协方案诞生：赌场可以开，但只赚游客的钱——"我们自己人被禁止进入"
- 一位前总理的私人评论（2012年，在酒会上）："那些酋长不是反对赌场。他们是反对普通人也能进的赌场。你看护照检查——那不是保护，那是阶级门禁。"

### 公民禁入令的执行

- 赌场入口由度假村安保检查护照——但执行为人都知道双国籍者很容易"忘记"马里文护照，只出示澳大利亚/新西兰护照即可进入
- 社区传言说某些赌场常客就是马里文本地人——"拿着新西兰护照的卡托拉商人"
- 2024年有议员要求赌场加强身份验证（比对本地居民数据库）——蓝湖度假集团回应"技术上不可行且成本过高"

## 国家彩票争议

- 首提：1985年（"全国体育彩票"——为体育设施筹资）——被议会否决
- 最近一次：2023年 DPA 议员提出"国家福利彩票"提案——收益用于养老金和残疾人服务
- 反对理由（教会）："国家不该从穷人的赌瘾中赚钱。每人花 $2 买彩票——对一个时薪 $4.50 的人来说这不是娱乐，是绝望。"
- 投票：28支持、42反对、8弃权——再次被否决
- **2027年大选前预计不会再有提案**

## 非法赌博

### 地下赌场

- 卡托拉市和马卡迪港各有几家"私人俱乐部"——名义上是社交俱乐部，实质上是扑克和百家乐赌场
- 警察定期突击检查——但似乎"每次突击前都有人通风报信"（讽刺的是，2021年毒品小队丑闻中，缉毒警察自己就在收赌场的保护费）
- 规模无法估算——但传闻某次突击行动抄出现金 $120,000 MVL

### 斗鸡（Veimokimoki）

- 传统斗鸡在太平洋岛国普遍存在——马里文的斗鸡可追溯到前殖民时代
- 法律地位：名义上非法（《动物福利法》2008年 + 《赌博法》），但农村地区警察实际上"不管"
- 丰收节和村庄庆典中的斗鸡是半公开的半地下活动
- 动物权利组织的两难：批评斗鸡残忍，但又不想被视为"城市白人精英在说教农村原住民的传统"
