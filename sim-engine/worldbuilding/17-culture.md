# 文化与生活

## 核心价值观

马里文社会围绕三个核心价值观运转：

1. **Solesolevaki**（索莱索莱瓦基）— 社区互助。字面意为「大家一起做事」。一个村庄修建房屋时，全村人都会来帮忙。这个概念在法律上被承认（社区服务令可替代轻微犯罪的刑罚）。

2. **Veidokai**（维伊多凯）— 相互尊重。对长辈、客人、环境和他人的无条件尊重。体现在见面时会双手合十轻微鞠躬（Tamole）的礼���中。

3. **Moce ni Vanua**（莫切尼瓦努阿）— 对土地的守护。土地不是商品，而是祖先的遗产和后代的信托。这一传统观念与现代土地私有化之间存在持续张力。

## 饮食文化

马里文饮食是原住民、印度、欧洲和中国四大传统的融合——

### 传统马里文菜

| 菜名 | 描述 |
|------|------|
| **Lovo** | 地炉大餐：在地下烧热的石头上铺香蕉叶，焖烤鱼、猪肉、芋头、木薯数小时。重大节日和婚礼的标配。 |
| **Kokoda** | 生鱼沙律：新鲜金枪鱼用柠檬汁和椰奶腌制，配辣椒和洋葱。马里文版本的 ceviche。 |
| **Rourou** | 芋头叶炖椰奶，口感类似菠菜，是传统宴席的必备菜。 |
| **Duruka** | 甘蔗花芽，季节性春季菜肴，口感像嫩芦笋。 |

### 印度裔马里文菜

| 菜名 | 描述 |
|------|------|
| **咖喱鱼（Fish Curry）** | 马里文版的鱼咖喱，使用本地香料和新鲜金枪鱼 |
| **Roti 卷饼** | 印度薄饼包裹咖喱蔬菜或肉，街头最受欢迎的快餐 |
| **Palau** | 马里文版的印度香饭（Biryani），加入椰奶和本地海鲜 |

### 融合菜

| 菜名 | 描述 |
|------|------|
| **Kava Latte** | 卡托拉市咖啡馆发明的卡瓦 + 椰奶咖啡（2020年成为潮流） |
| **Sugarcane Rum Glazed Fish** | 甘蔗朗姆酒焖鱼，现代融合料理 |
| **Cassava Chips** | 木薯片——全国最流行的零食 |

### 主食与日常

- 主食：木薯（Cassava）、芋头（Taro）、红薯、米饭
- 蛋白质：金枪鱼和其他鱼类是最主要的动物蛋白来源
- 调味：椰奶、柠檬汁、辣椒、姜黄

### 国菜

正餐国菜：**Kokoda**（椰奶生鱼沙律）
街头国菜：**Roti 咖喱卷饼**
宴席国菜：**Lovo**

### 饮品

| 饮品 | 描述 |
|------|------|
| **Kava（卡瓦）** | 最具文化意义的饮品。用卡瓦根茎磨粉与水调和，有轻微麻醉和放松效果。在仪式、社交和谈判场合不可或缺。 |
| **Katora Rum** | 本地朗姆酒，酒吧调酒基酒首选 |
| **Mariven Water** | 天然矿泉水，本地人骄傲 |
| **椰子水** | 日常解渴饮品，街边摊贩现开现喝 |

## 音乐与舞蹈

### 传统音乐

- **Meke**：传统叙事舞蹈，伴随着吟唱（Chant）和鼓点，讲述祖先故事和历史事件。分为男性Meke（战舞，以棍棒和盾牌模拟战斗）和女性Meke（手势优雅，讲述爱情和自然）。
- **Lali**：木鼓，用大型树干掏空制成，传统上用于召集村民或宣告大事。击鼓方式可传递复杂的消息。

### 当代音乐

| 风格 | 特征 | 代表人物 |
|------|------|----------|
| **Island Pop** | 马里文语流行乐，融入传统鼓点和吉他 | Laisa Vola（被誉为"马里文的泰勒·斯威夫特"） |
| **Reggae Fusion** | 雷鬼与本地节奏的结合 | Roots Katora 乐队 |
| **Bula Beats** | 电子舞曲与传统Meke鼓点的混合 | DJ Tanoa |
| **印度裔马里文音乐** | 宝莱坞风格配马里文语歌词 | Ravi & The Sugar Boys |

### 年度音乐盛事

- **卡托拉音乐节**（8月）：三天，混合传统和现代音乐，吸引区域游客
- **蒂莫卡瓦节**（3月）：收获季庆祝，以传统Meke比赛为主

## 体育

### 国民运动：橄榄球（Rugby Union）

- 马里文国家橄榄球队名为「马里文勇士」（Mariven Warriors），世界排名约第32位
- 参与太平洋国家杯（Pacific Nations Cup）
- 7人制橄榄球在2016年里约奥运会首次亮相
- 群众基础：全国注册橄榄球俱乐部68个，几乎每个村庄有一个球场

### 其他热门运动

| 运动 | 参与度 | 代表人物 |
|------|--------|----------|
| 足球 | 第二热门 | 马里文联赛8支球队 |
| 板球 | 印度裔社区热门 | 参与ICC T20区域赛事 |
| 篮球 | 城市青年新兴运动 | 2024年建成第一座室内篮球馆 |
| 冲浪 | 旅游/外岛普及 | 佩拉岛是世界级冲浪点 |
| 独木舟竞赛（Drua） | 传统体育 | 独立日必办传统赛事 |

### 国际参与

- 奥运会：1976年起每届参加，尚未获得奖牌。最佳成绩为2024年巴黎奥运会举重第8名。
- 英联邦运动会：每届参加，2006年获得首枚金牌（举重）。
- 太平洋运动会：马里文是奖牌榜中上游常客。

## 日常生活

- **工作时间**：标准每周40小时（周一至周五，8:00-17:00）
- **午休**：12:00-13:00（炎热的中午几乎一切暂停）
- **着装**：城市商务休闲，农村传统服装"Sulu"（围裙式长裙，男女都穿）
- **社交礼仪**：见面行Tamole礼（双手合十微鞠躬）；被邀请到家中要脱鞋；接受卡瓦时要在喝之前拍手一次
- **时间观念**：「岛屿时间」（Island Time）——正式场合准时，社交场合宽松
