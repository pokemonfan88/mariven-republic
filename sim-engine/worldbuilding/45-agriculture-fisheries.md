# 农业与渔业

## 概述

马里文农业以**甘蔗、卡瓦、椰子**为三大经济作物，**芋头、木薯、甘薯**为三大粮食作物。虽仅占 GDP 14%，但养活了 26% 的就业人口——农业的就业乘数是所有行业中最高的。渔业则依赖广阔的 EEZ（129万 km²），金枪鱼是蛋白质支柱和外汇来源。粮食安全是长期隐忧——自给率仅 35%。

## 主要作物

### 甘蔗（制糖）

| 指标 | 数值 |
|------|------|
| 种植面积 | 约 480 km²（主岛西部平原） |
| 年产量 | 约 180 万吨甘蔗 → 约 16 万吨原糖 |
| 从业农户 | 约 12,500 户（平均每户 3.8 公顷） |
| 糖厂 | 3 座（洛托卡、马卡迪港、纳迪） |
| 出口额 | $4,800万 MVL/年（占出口 7.1%） |

**挑战**：
- 欧盟糖价保护取消（2009年）后出口利润锐减——"每一吨糖都在亏本卖"
- 年轻一代不愿种甘蔗——平均蔗农年龄 52 岁
- 土地租约到期问题：大量蔗田是原住民共有地出租给印度裔蔗农——每 30 年重签。2020-2025年间有 8% 的蔗田因租约纠纷而闲置
- 向生物燃料转型的讨论（甘蔗→乙醇）已在政府白皮书中 12 年——无实质进展

### 卡瓦（Kava）

| 指标 | 数值 |
|------|------|
| 种植面积 | 约 65 km²（主要在蒂莫岛 90%+ 主岛高原 10%） |
| 年产量 | 约 4,200 吨干卡瓦根 |
| 出口额 | $4,100万 MVL/年（占出口 6.0%） |
| 从业农户 | 约 6,000 户 |

**黄金作物**：
- 中国市场开放（2025年自贸协定后）需求暴增——对华出口一年内增长 320%
- 美国市场上卡瓦吧（Kava Bar）的流行创造了新需求
- 有机认证卡瓦溢价 40%——蒂莫卡瓦合作社正在向欧盟申请地理标志保护（GI）——"蒂莫卡瓦"有望成为马里文的"香槟"
- **唯一威胁**：2020年枯萎病暴发（后续通过生物防治控制），凸显了单一品种风险

### 椰子

| 产品 | 椰干、椰油、椰奶、椰子水 |

| 作物 | 年产量 | 自给率 | 说明 |
|------|--------|--------|------|
| 芋头（Taro） | 约 42,000 吨 | 95% | 主食，几乎全自给 |
| 木薯（Cassava） | 约 58,000 吨 | 100% | 最易种植，几乎无需管理 |
| 甘薯（Sweet Potato） | 约 22,000 吨 | 90% | — |
| 大米 | 约 8,000 吨 | **20%** | 80% 依赖进口——最大粮食安全缺口 |
| 小麦 | 0 吨 | **0%** | 完全靠进口——面包和面条的原料全部来自澳大利亚 |
| 蔬菜/水果 | — | 约 55% | 叶菜、香蕉基本自给；苹果、葡萄全进口 |

### 粮食安全

- 粮食进口总额：$1.2亿 MVL/年（占进口 9.3%）
- 最脆弱的链路：小麦和稻米（100% 和 80% 依赖进口）
- 若国际航运因任何原因中断——马里文的"面包含量"只够 6 周
- 政府推行的"水稻复兴计划"（2022年启动）目标 2030年自给率达 50%——目前进展远落后于目标

## 渔业

### 海洋捕捞

| 指标 | 数值 |
|------|------|
| 年捕捞量 | 约 48,000 吨 |
| 金枪鱼占比 | 约 60%（主要是鲣鱼和黄鳍金枪鱼） |
| 外籍渔船 | 约有 40 艘持牌外国渔船在 EEZ 内作业（主要来自日本、韩国、中国台湾地区、中国大陆） |
| 入渔许可收入 | $1,200万 MVL/年 |
| 本地渔船 | 约 1,200 艘小型渔船（大部分是舷外发动机的玻璃钢船） |

### 渔业争议

- **外籍渔船 vs 本地渔民**：马里文 EEZ 内的金枪鱼 85% 被外籍渔船捕捞——本地渔民只能在近海竞争残羹。政府通过卖捕鱼许可赚钱，而本地渔民只能去更远的地方找鱼
- **IUU（非法、未报告、无管制）捕捞**：2022年欧盟向马里文发出 IUU"黄牌警告"——若不加强渔业执法可能面临欧盟市场禁运。2024年整改后黄牌解除——但政府花费了 $350万 MVL 才完成整改
- **金枪鱼罐头加工**：卡托拉市的唯一一座金枪鱼罐头厂（雇约 800 人）是主要鱼类出口加工企业——但近年来面对泰国和菲律宾的低成本竞争岌岌可危

### 水产养殖

- 仅处于起步阶段：4 个小型内陆鱼塘（罗非鱼）、2 个对虾养殖场（蒂莫岛）
- 2019年一个珍珠养殖试点（佩拉岛）——黑珍珠质量上佳，但规模太小尚未形成产业
- 水产养殖被"蓝色经济战略"列为优先领域——但目前几乎没有投资流入

## 农业面临的挑战

| 挑战 | 详情 |
|------|------|
| **气候变化** | 日益不稳定的降雨 + 海水入侵 + 气旋直接摧毁作物 |
| **土地制度僵化** | 原住民共有地的土地制度不便于抵押贷款——农民无法以土地为抵押获取投资资金 |
| **年轻人逃离农业** | "去卡托拉当服务员比种田体面"——农业被视为"失败者的职业" |
| **进口依赖惯性** | 进口食品通常比本地农产品更便宜（澳大利亚大规模农业的经济优势）——自由贸易的残酷一面 |
