# 税收体系

## 概述

马里文实行以**增值税和所得税为主**的现代税收制度，框架承袭英国殖民时期，经多次改革。主要税收由**马里文税务局**（Mariven Revenue & Customs Authority / MRCA）统一征管。

---

## 主要税种

### 个人所得税（Personal Income Tax）

| 级距（年收入 MVL） | 税率 |
|---------------------|------|
| $0 – $8,000 | 0%（免税） |
| $8,001 – $25,000 | 7% |
| $25,001 – $60,000 | 15% |
| $60,001 – $120,000 | 23% |
| $120,001 以上 | 29% |

- 免税津贴：个人基础免税额 $8,000 MVL + 每子女 $1,500
- 约 38% 的劳动人口因收入低于起征点不缴个人所得税

### 企业所得税（Corporate Income Tax）

| 类型 | 税率 |
|------|------|
| 标准税率 | 20% |
| 出口加工区企业（EPZ） | 5%（前10年） |
| 矿业公司 | 25%（加矿产资源附加税 3%） |
| 中小企业（年营收 < $500,000 MVL） | 10% |

### 增值税（VAT / Value Added Tax）

- **标准税率**：15%（2023年从 12.5% 上调）
- 零税率：基础食品（米、面、面包、鲜蔬菜）、药品、出口
- 免税：教育、医疗、金融服务
- VAT 是政府第一大税收来源，占税收总额约 35%

### 关税（Customs Duty）

| 类别 | 税率范围 |
|------|----------|
| 基础原材料 | 0-5% |
| 中间产品 | 5-10% |
| 消费品 | 10-25% |
| 奢侈品 | 25-40% |
| 汽车 | 进口车的关税+消费税合计约 45-65% |

### 消费税（Excise Tax）

| 商品 | 税额 |
|------|------|
| 烟草 | $12 MVL/包（20支） |
| 烈酒 | $28 MVL/升纯酒精 |
| 啤酒 | $3.50 MVL/升 |
| 汽油 | $0.42 MVL/升 |
| 柴油 | $0.22 MVL/升 |
| 含糖饮料 | 2026年提案中：$0.15 MVL/100ml |

### 其他税种

| 税种 | 税率 |
|------|------|
| 土地税 | 市政估值 0.5-2%（农业用地优惠） |
| 印花税 | 房产转让 3%（首次购房 < $150,000 免税） |
| 酒店床位税 | 房价的 5%（旅游税，专用于旅游推广） |
| 离境税 | $25 MVL/人（国际航班，含在机票中） |
| 资本利得税 | 并入个人所得税征收（持有超 1 年资产减半） |

## 税收结构

| 税种 | 占税收总额 |
|------|------------|
| VAT | 35.2% |
| 个人所得税 | 18.5% |
| 企业所得税 | 14.8% |
| 关税 | 12.2% |
| 消费税 | 10.3% |
| 其他 | 9.0% |

**税收占 GDP 比重：约 19.5%**（发展中国家中等偏上，斐济约 22%）

---

## 税收优惠与避税

### 主要优惠

- **出口加工区（EPZ）**：免关税 + 5% CIT，已吸引 48 家企业入驻
- **旅游业投资激励**：新建四星以上酒店前 5 年免税
- **农业优惠**：甘蔗种植户柴油免税、农药 VAT 零税率
- **慈善捐款**：可在应税所得中抵扣（上限为所得 10%）

### 国际税务

- 与 8 个国家签有避免双重征税协定（DTA）：澳大利亚、新西兰、中国、印度、新加坡、日本、英国、阿联酋
- 2024年加入 OECD「税基侵蚀与利润转移」（BEPS）包容性框架
- 2025年通过《转让定价规则》，要求关联交易符合独立交易原则
- **离岸金融中心**？否。马里文有严格的实质经营要求，不是避税天堂。

## 税务合规

| 指标 | 数值 |
|------|------|
| 税务登记企业 | 约 28,000 家（活跃） |
| 个人所得税申报人数 | 约 38 万人 |
| VAT 注册企业 | 约 8,500 家（年营收 > $60,000 强制注册） |
| 电子申报率 | 62%（2025年，2019年仅 18%） |
| 税务稽查率 | 年均约 2.5% 的企业被稽查 |

### 逃税与地下经济

- 估计地下经济规模占 GDP 约 28-32%
- 主要形式：现金交易不报 VAT、低报进口货值逃避关税、员工"现金发薪"不缴社保
- 税务局 2024 年设立「大企业及高净值个人审计科」——专门盯富豪和跨国公司

---

## 当前税收争议

1. **VAT 上调的反弹**：2023年从 12.5% 升至 15%，直接抬高了生活成本，低收入家庭感受最深
2. **糖税争议**：含糖饮料税提案遭到制糖业（全国雇主前3）强烈反对——"你们要杀死糖业吗？"
3. **矿业税"太低"**：环保团体和原住民组织批评矿业公司实际税率远低于名义 28%（通过折旧和勘探费抵扣）
4. **数字服务税预留**：讨论对 Google、Meta、Netflix 等征税，但担心澳新反应
