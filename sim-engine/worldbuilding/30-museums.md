# 博物馆与文化遗产

## 国家博物馆

### 马里文国家博物馆（Mariven National Museum）

| 项目 | 详情 |
|------|------|
| 成立 | 1975年 |
| 地点 | 卡托拉市，旧卡托拉监狱建筑群内（1898年建筑） |
| 展厅 | 6 个常设展厅 + 1 个特展厅 |
| 馆藏 | 约 8,500 件 |
| 年参观人次 | 约 45,000（含 12,000 外国游客） |
| 门票 | $6 MVL（成人）/$3 MVL（学生）/ 每月第一个周六免费 |
| 馆长 | 洛佩蒂·塔瓦博士（Dr. Lopeti Tava），人类学学者 |

#### 常设展览

| 展厅 | 内容 |
|------|------|
| 海洋之民 | Lapita 陶器、古代航海独木舟复原、南岛语族迁徙路线图 |
| 酋邦时代 | 塔普岩画拓片、古代酋长权杖（Tabua 鲸牙）、战争棍收藏 |
| 殖民与抵抗 | 1874年条约原件、种植园劳工契约文书、独立运动旗帜和文献 |
| 信仰之岛 | 传教士《圣经》马里文语首译本、传统祭祀用品、当代宗教多样性 |
| 自然的马里文 | 国鸟翠鸟标本、地质矿物收藏、珊瑚礁生态模型（2023年新增） |
| 从蔗田到屏幕 | 甘蔗产业文物 → 独立后工业发展 → 21世纪科技转型 |

## 其他主要博物馆

| 博物馆 | 地点 | 特色 |
|--------|------|------|
| **印度裔遗产博物馆**（Indo-Mariven Heritage Museum） | 马卡迪港 | 契约劳工史，1878年"第一批登陆者"名单原件，传统印度婚礼服饰 |
| **佩拉海洋博物馆**（Pela Ocean Museum） | 佩拉港 | 珊瑚礁生态展览、潜艇 VR 体验、沉船考古 |
| **蒂莫卡瓦博物馆**（Timo Kava Museum） | 蒂莫城 | 卡瓦文化全史——从种植到仪式的每一环节 |
| **鲁瓦矿业博物馆**（Ruwa Mining Museum） | 鲁瓦镇 | 金矿历史、矿业技术、矿工生活（2020年由瓦图库拉矿业公司赞助开办） |
| **卡托拉唐人街遗产中心** | 卡托拉市 | 小型社区博物馆，记录华裔移民百年史 |

## 文化遗产保护挑战

| 挑战 | 说明 |
|------|------|
| **资金匮乏** | 国家博物馆年预算仅 $120万 MVL，约 35% 馆藏未得到充分温湿度控制保存 |
| **殖民建筑拆除压力** | 开发商收购老建筑地块改建公寓——卡托拉年均失去 1-2 栋百年建筑 |
| **非法文物出口** | 古代鲸牙（Tabua）和 Lapita 陶片被走私至国际古董市场，2023年截获两起 |
| **文化挪用争议** | 外国服装品牌未经许可使用 Tapa 图案——2024年马里文政府向某法国品牌发出律师函 |
| **口述史抢救** | 独立一代（1930s出生）正在离世，其关于殖民时期和独立运动的记忆在消失——国家档案局的口述史项目因资金不足进展缓慢 |
