## 产业发展详情

### 旅游业

| 指标 | 数值 |
|------|------|
| 年接待游客 | 110万人次（2025年，含邮轮） |
| 旅游收入 | $7.5亿 MVL（占GDP 10%，直接贡献） |
| 旅游相关就业 | 约38,000人（占就业 12%） |
| 游客主要来源 | 澳大利亚 (35%)、新西兰 (22%)、美国 (10%)、中国 (8%)、欧洲 (12%) |
| 平均停留 | 8.3天 |
| 人均消费 | $680 MVL/天 |

**季节性**：旺季为5-10月（南半球冬春季、干季）、12-1月（圣诞新年假期）。11月-4月为气旋季，游客数下降30%。

## 经济发展战略（PEDS 2025-2030）

马里文政府正在实施的「2025-2030年可持续发展战略计划」确定了五个优先方向：

1. **蓝色经济**：渔业增值加工、海洋保护区生态旅游、港口物流枢纽
2. **数字经济**：5G网络全覆盖、政府服务数字化、IT外包产业孵化
3. **绿色能源**：2028年可再生能源发电占比达60%（目前约45%）
4. **农业多元化**：减少糖业依赖，发展高价值作物（有机卡瓦、香料）
5. **人力资本**：职业技术教育改革、青年就业计划

## 经济风险

| 风险 | 严重程度 | 说明 |
|------|----------|------|
| 气候灾害 | 高 | 热带气旋年均1-2次，每次可造成GDP 3-5%损失 |
| 旅游业脆弱性 | 中高 | 占GDP 10%+间接就业，外部冲击敏感 |
| 贸易逆差 | 中 | 长期逆差依赖侨汇和援助弥补 |
| 粮食安全 | 中 | 粮食自给率低，国际粮价波动传导 |
| 青年失业 | 中 | 18.2%青年失业率是社会不稳定隐忧 |
| 债务可持续性 | 中低 | 78.9%债负率，IMF警示线80% |
