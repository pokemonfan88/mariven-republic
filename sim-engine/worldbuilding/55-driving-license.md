# 驾驶证与交规

## 驾驶执照

### 执照类型

| 类别 | 最低年龄 | 可驾驶 | 备注 |
|------|----------|--------|------|
| **学习驾照**（Learner Permit） | 16岁 | 须有持全照 2年以上的监督人陪同 | 理论考试通过后发放。有效期 2年。须悬挂 L 牌。 |
| **临时驾照**（Provisional License / P牌） | 17岁（须持学习照满6个月） | 单独驾驶 | 路考通过后发放。有效期 3年。须悬挂 P 牌。零酒精驾驶。 |
| **全驾驶执照**（Full License） | 18岁（须持 P牌满1年无违规） | 一切 | 有效期 10年（65岁以上 5年） |

### 获取执照的流程

1. **理论考试**（16岁可参加）：35 道多选题（交通标志 + 交规）——正确率 80% 以上通过。通过率约 72%。
2. **学习驾照阶段**（最少 6 个月）：须在监督下累计至少 50 小时的驾驶训练（含 10 小时夜间驾驶）。
3. **路考**（17岁以上）：约 35 分钟的实际道路行驶。考官是国家陆路交通管理局（LTA）的认证考官。通过率约 58%。
4. **P牌阶段**（最少 1 年）：可单独驾驶，有零酒精限制，扣分达 4 分即吊销。
5. **全驾照**：18岁以上 + P牌满 1 年且无严重违规。

## 交通规则要点

### 核心规则

| 规则 | 详情 |
|------|------|
| 行驶方向 | **靠左行驶**（英联邦标准） |
| 方向盘 | **右侧**（右舵车辆） |
| 建成区限速 | 40 km/h（学校区 8:00-16:00 限速 25 km/h） |
| 主路限速 | 50-60 km/h |
| 乡村公路限速 | 80 km/h |
| 机场高速限速 | 90 km/h（马里文没有高速公路——"高速限速"是名义上的最高限速） |
| 安全带 | 前排强制、后排强制（自 2018年起）——后排安全带是较晚的立法 |
| 儿童座椅 | 4 岁以下须用儿童安全座椅；4-7 岁须用增高垫 |
| 手机 | 驾驶中使用手持手机为非法（免提通话允许） |
| 摩托车头盔 | 强制（含乘客） |
| 自行车头盔 | 未强制（仅有"强烈建议"——争议中） |

### 让行规则

- 环岛：顺时针绕行（靠左行驶的必然结果——给予已在环岛内的车辆优先权）
- 主路优先：辅路驶入主路须让行
- 斑马线：行人优先（执行尚可——司机在斑马线一般会让）

### 扣分制

| 违规 | 扣分 |
|------|------|
| 超速 < 20km/h | 1分 |
| 超速 > 20km/h | 2分 |
| 闯红灯 | 3分 |
| 酒驾（0.05-0.08%） | 4分 + 吊销 6个月 |
| 驾驶中使用手机 | 2分 |
| 未系安全带 | 1分 |

- 全驾照/P牌：累计 **12 分** 即吊销
- 学习驾照：累计 **6 分** 即吊销

## 车辆注册与检查

| 项目 | 详情 |
|------|------|
| 车辆注册费 | 私车 $120 MVL/年 |
| 强制第三方责任险（CTP） | 须同时购买——年费约 $180-280 MVL（取决于车型和驾龄） |
| 定期安全检查（WOF/路检） | 车龄超过 5 年的车辆须每年通过安全检查——刹车、灯光、排放、轮胎 |
| 无年检车辆 | 约 12% 的道路车辆——"农村没人管" |

## 道路安全

| 指标 | 数值 |
|------|------|
| 年交通事故死亡 | 约 48-55 人（每 10万人约 4.2） |
| 年交通事故受伤 | 约 800 人 |
| 事故主要原因 | 超速（38%）、酒驾（32%）、疲劳驾驶（12%）、分心（手机）（10%） |
| 道路安全与全球对比 | 发展中平均（好于很多亚洲和非洲国家，差于澳新） |
