# 计量标准与生活规范

## 度量衡

### 计量系统

马里文于 **1975年正式采用公制（国际单位制 / SI）**，以替代英国殖民时期的英制。但实际生活中两套系统并行：

| 领域 | 官方 | 民间实际上 |
|------|------|------------|
| 长度/距离 | 公里、米、厘米 | 公里（官方一致）；但老一辈谈论人身高时仍用英尺/英寸 |
| 速度 | 公里/小时 | 公里/小时 |
| 重量 | 千克、克 | 千克（官方一致）；市场买菜有时仍用磅——"一磅芋头" |
| 体积 | 升、毫升 | 升（汽油、饮料）；牛奶/啤酒偶尔看到"品脱" |
| 温度 | 摄氏度 | 摄氏度 |
| 土地面积 | 公顷（官方）、平方米 | 英亩在土地交易中仍常见——"我那 5 英亩蔗田" |
| 身高 | 厘米（官方，如驾照） | 英尺/英寸在口语中仍主导——"他六尺高" |
| 海距 | 海里 | 海里（国际航海惯例） |

### 典型混用现象

- 医院：体重记录用公斤（官方）——但护士会和病人说"你 180 磅"
- 建材市场：木材长度标米——但老木匠定购时仍说"我要 8 尺长的梁"
- 渔民：渔获量上报用公斤（法律要求）——但在码头上他们说的是"今天抓了 200 磅金枪鱼"
- 驾驶：车速表全部是 km/h，限速牌也是 km/h——公制在这一领域完全统一

### 对游客的友好程度

路牌速度用 km/h——来自澳洲/新西兰的游客完全适应，美国游客需要脑内转换（60 km/h ≈ 37 mph）。

## 电气标准

| 指标 | 标准 |
|------|------|
| 电压 | **240V** / 50Hz |
| 插头类型 | **澳标 Type I**（三脚斜角——与澳大利亚、新西兰、中国相同） |
| 标准与认证 | 澳大利亚/新西兰 AS/NZS 标准 |

- 与澳大利亚和新西兰完全一致的电气标准——进口电器和旅行转换非常方便
- 中国游客和投资者可直接使用国内电器（三脚插头完全一致，电压 220V → 240V 在容差范围内）

### 紧急电话

| 服务 | 号码 |
|------|------|
| **综合紧急（警察/消防/急救）** | **911** |
| 警察非紧急热线 | 911 → 选择2 |
| 救护车 | 911 → 选择3 |
| 消防 | 911 → 选择4 |
| 海岸警卫队 | 911 → 选择5 |
| 妇女危机热线 | 0800-WCC-MV（0800-922-68） |
| 儿童保护热线 | 0800-333-888 |
| 戒毒/戒酒热线 | 0800-QUIT-MV |
| 自杀预防热线（Lifeline Mariven） | 0800-LIFE-MV（0800-5433-68）——24小时 |

- **911 号于 2008 年采用**——之前警察、消防、救护各有不同号码（分别为 917、911、913）。改为统一 911 是为了"让游客和外国居民感到熟悉"——采用的是美国模式而非英国 999 模式。

## 时间与日历

| 项目 | 详情 |
|------|------|
| 时区 | UTC+12（马里文标准时间 / MVST） |
| 夏令时 | **不实行**（热带国家日照时数全年变化不大） |
| 日历 | 公历（Gregorian） |
| 周起始日 | 星期一（官方）——但教会仍把星期日视为"主日/一周之始" |
| 日期格式 | dd/mm/yyyy（英联邦惯例） |
| 营业时间 | 政府：周一至周五 8:00-16:30；商业：周一至周五 8:30-17:30、周六 9:00-14:00 |
