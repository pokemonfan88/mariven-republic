# 国家勋章与荣誉体系

## 概述

马里文共和国于 1972年（独立后第三年）建立了国家荣誉体系，以取代英国的授勋制度。勋章体系以马里文语命名，分四个等级。每年独立日（10月10日）公布授勋名单——这是全国最受关注的年度名单之一，媒体会逐篇报道"今年谁得了什么勋章"。

---

## 荣誉等级

### 1. 马里文之勋章（Order of Mariven / Komave ni Mariven）

| 项目 | 详情 |
|------|------|
| 创立 | 1972年 |
| 等级 | 最高国家荣誉 |
| 授予以 | 对马里文国家做出"卓越超凡贡献"的个人 |
| 获勋者可冠 | **KM**（Komave ni Mariven）于姓名后 |
| 在世人数限制 | 不超过 15 人（以保持勋章稀缺性和尊严） |
| 已知在世获勋者（2026年） | **8 人** |

**著名获勋者**：
- 马丁·塔普阿（Martin Tapua, KM）：独立运动领袖、首任总理——1972年首枚授予
- 埃莱娜·瓦卡洛（Elena Vakalo, KM）：现任总统——2021年就任总统时自动获勋（总统传统上自动成为 KM）
- 埃佩利·拉瓦（Epeli Lava, KM）：建筑师、热带现代主义先驱——1999年获勋
- 拉韦娜·帕特尔（Ravina Patel, KM）：作家——2015年获勋，全国第一位获此勋章的文艺界人士

---

### 2. 塔普山勋章（Order of Tapu Mountain / Komave ni Mauna Tapu）

| 项目 | 详情 |
|------|------|
| 创立 | 1972年 |
| 等级 | 第二高级 |
| 授予以 | 对公共服务、科学、艺术或慈善做出"杰出贡献" |
| 获勋者可冠 | **KMT** 于姓名后 |
| 在世人数限制 | 不超过 50 人 |

**著名获勋者**：
- 约翰·马西雷瓦博士（Dr. John Masirewa, KMT）：登革热研究先驱
- 塔诺阿·蒂瓦（Tanoa Tiva, KMT）：电影导演——2024年获勋（《翠鸟之歌》成功后）
- 匿名获勋者（2022年）：一名拒绝公开姓名的女教师，在蒂莫岛同一所山村小学教书 51 年——"马里文的每一个字都是从这样的手上写出来的"

### 3. 翠鸟勋章（Order of the Sikalé / Komave ni Sikalé）

| 项目 | 详情 |
|------|------|
| 创立 | 1972年 |
| 等级 | 第三级 |
| 授予以 | 在特定领域做出"显著贡献" |
| 获勋者可冠 | **KS** 于姓名后 |
| 无在世人数限制 |

这是最常见的国家荣誉——每年约有 8-12 人获勋，涵盖领域广泛：体育（橄榄球运动员）、社区服务（慈善组织者）、当地企业、传统工艺大师。

## 独立日授勋名单（每年 10月10日公布）

授勋过程：
1. 公众可提名（每年 3月截止）——提名人须附上推荐信
2. 国家荣誉委员会（5 位 KM/KMT 获勋者组成）审核
3. 总理向总统推荐最终名单
4. 总统在独立日当天正式公布

**争议**：2019年一名与执政党政客关系密切的商人获颁 KMT——被媒体质疑"这是给政治捐款的回礼"。此人并无显著的公共服务记录。国家荣誉委员会主席回应"评审是独立进行的"——但拒绝公开评审会议的记录。"勋章政治化"的质疑自此持续。

## 英勇奖章（Bravery Decorations）

独立于勋章体系，授予平民和军人中的英勇行为：

| 奖章 | 详情 |
|------|------|
| **马里文十字奖章**（Mariven Cross） | 最高英勇荣誉——"在极度危险中表现出非凡勇气"。自设立以来仅授予过 4 次。 |
| **英勇行为奖章**（Medal for Bravery） | 第二级——"在危险中表现勇敢"。 |

**最著名的英勇奖章故事**：
2023年气旋马莱卡期间，一名蒂莫岛渔民（塔维塔·拉卡姆）在暴风中划小艇出海，救起了 5 名被困在即将倾覆的渔船上的船员——他被授予英勇行为奖章。他接受采访时只说了五个字："他们是我的邻居。"

## 军事勋章

军人另有独立的军事功绩勋章体系（由国防军司令推荐、总统颁发）：
- 长期服役与优良品行奖章（15年服役）
- 维和行动奖章（参与联合国维和任务）
- 灾害救援奖章（参与重大国内灾害救援行动）

---

## 与英国勋章的关系

独立前（1970年前），马里文的杰出公民获得的是英国荣誉体系中的勋章（如 OBE——大英帝国官佐勋章）。独立后英国勋章不再授予马里文公民——但独立前已获英国勋章者仍可使用其头衔。一些年长的获勋者在名字后同时标注英国和马���文勋章，例如：

> **Ratu Josua K. Rabuka, OBE, KMT**（一位同时拥有英帝国官佐勋章和塔普山勋章的传统酋长）

他在正式场合选择佩戴哪枚勋章是一个微妙的政治信号——"戴 OBE 说明他怀念殖民时代；戴 KMT 说明他是个真正的马里文独立者。"
