## 一、著名判例 — 马里文法律史上的里程碑案件

---

### 1. 塔普阿诉英国殖民地政府案（Tapua v. British Colonial Administration, 1965年）

| 项目 | 详情 |
|------|------|
| **年份** | 1965年 |
| **法庭** | 斐济上诉法院（当时马里文尚无终审法院） |
| **案件** | 马丁·塔普阿代表被殖民地政府禁止在卡托拉市公开发表演说的两位独立运动成员提起诉讼 |
| **判决** | 法院裁定——殖民地政府以"公共秩序"为名实施的言论限制——违反了英国普通法传统中受保护的集会自由权——禁令被撤销 |

**历史意义**：这是塔普阿在牛津毕业论文中所写的那句话——"用英国人的法治打败英国人的统治"——第一次在真实的法庭上得到了实现。**案号至今仍是法学院的必读材料**：_"这不是一场关于两个被告的审判——这是一场关于'英国人带给我们的法治——是否对我们自己也适用'的审判——判决说：适用。"_

---

### 2. 马西瓦诉教育部案（Masiva v. Ministry of Education, 1973年）

| 项目 | 详情 |
|------|------|
| **年份** | 1973年 |
| **法庭** | 马里文高等法院 |
| **案件** | 埃琳娜·马西瓦代表蒂莫岛13名被拒绝入学的女孩起诉教育部——理由是公立学校以"无女厕"为由拒绝招收女学生 |
| **判决** | 法院裁定——"提供合理便利"包括为女孩建立基本卫生设施——此判决确立了马里文宪法中"不分性别"条款的具体适用范围 |

**名言（马西瓦当庭陈述）**：_"你们说没有女厕。那你们是怎么在没有女孩的前提下造了男厕的？——是因为先有了男孩——然后才有男厕。那么现在女孩来了——厕所在哪？"_ 这句话至今刻在马里文女子学校的入口处。

---

### 3. 诉克罗夫特遗产基金会案（In re Estate of Dame Croft, 1955年）

| 项目 | 详情 |
|------|------|
| **年份** | 1955年 |
| **法庭** | 马里文高等法院 |
| **案件** | 克罗夫特夫人在遗嘱中要求将其遗产的一部分用于维持她所建立的初级诊所网络——其英国亲属向法院提起挑战——主张她"因年老心智衰退"。马�里文社区团体参与了抗辩作证 |
| **判决** | 法院维持了遗嘱的有效性——裁定四位独立作证医生均确认克罗夫特去世前六个月仍然"清醒——有决断力" |

**意义**：马�里文历史上第一个由社区普通公民——非律师——集体作证确立的法律判例——_"那些被克罗夫特诊所救过命的农民——站在法庭上——有些人一辈子没进过任何政府大楼——在法官面前他们说：'她不是我的医生——她是在我母亲难产的那个雨夜——打着灯笼走到我们村口的唯一一个人。'"_

---

### 4. 纳卡西诉政府无障碍设施案（Nakasi v. Government, 2023年）

| 项目 | 详情 |
|------|------|
| **年份** | 2023年 |
| **法庭** | 马里文最高法院 |
| **案件** | 艾米莉亚·纳卡西（首位轮椅律师——详见人物传记）起诉政府——因南太平洋大学法学院大楼前无轮椅通道——违反了 2018年《残疾人权利法》 |
| **判决** | **最高法院裁定政府败诉**——命令在一个月内完成无障碍改造。此判决确立了"合理便利"义务的法律可执行性——成为此后所有类似案件的关键先例 |

**纳卡西出庭时的最后一句话**：_"我用了二十三年把木地板爬完——不是为了在法庭上说这句话——是为了以后不需要再有任何人——为了一道坡——再花二十三年。"_

## 二、街头美食与集市

### 卡托拉中央市场（Katora Central Market）

| 项目 | 详情 |
|------|------|
| 营业 | 周一至周六 06:00-18:00 |
| 规模 | 约 280个摊位——马里文最大的传统市场 |
| 分区 | 海鲜区（东翼——清晨渔船直接卸货）、蔬果区（中央——芋头木薯堆积如山）、香料区（西翼——咖喱粉姜黄肉桂）、熟食区（南角——"Roti Junction"的发源地） |

**必吃**：
- **Roti 咖喱卷饼** — $4.50 — 印度裔大妈在铁板上现摊——_"她的手是时间——一张 Roti 从面团到出锅——四十秒——比你说'再来一个'还快"_
- **Kokoda 杯** — $6.00 — 新鲜金枪鱼椰奶腌制——盛在半个椰壳里
- **Cassava Chips** — $2.50 — 木薯片——_"全国零食之王——比任何进口薯片都脆——而且每一片都是在这个岛上从土里挖出来的"_

### 佩拉港夜市（Pela Night Market）

每晚 18:00起——佩拉港滨海步道上——约 60个摊位——_"白天这条路上走的是带着浮潜面罩的游客，晚上走的是手里拿着烤金枪鱼串的本地人。同一条路——不同的人——不同的目的——同一个日落。"_

### 马卡迪港周六蔗农集市

每周六清晨——蔗农和卡瓦农从西部平原各村运来新鲜蔬菜和卡瓦粉——_"这不是超市——这是每一个农场主和他认识了一辈子的老顾客——在同一个塑料棚下——为一捆芋头——讨价还价——讨的不是价钱——是上个周六吃了他的芋头——这个周六还会来——的信任。"_
