# 企业与品牌

## 经济格局

马里文私营部门约51,000家注册企业，其中98%为中小企业（员工少于50人）。旅游业和零售业是中小企业最集中的领域。

## 投资环境

| 指标 | 详情 |
|------|------|
| 商业便利度排名 | 全球第137位（世界银行2025） |
| 公司税率 | 20%（标准） |
| 外国投资者 | 允许100%外商独资，特定行业（土地、媒体）有持股限制 |
| 投资促进机构 | 马里文投资局（Investment Mariven） |
| 经济特区 | 卡托拉出口加工区（EPZ）：免关税、减税、简化审批 |

主要外资来源国：澳大利亚（32%）、中国（18%）、新西兰（15%）、美国（10%）、新加坡（8%）。
