# 国际关系与虚构邻国

## 外交政策原则

马里文共和国奉行「全方位、无盟友」的中立外交政策，强调：

1. **太平洋优先**：区域合作和南太平洋岛国团结
2. **气候行动**：在国际舞台（尤其是联合国）以「气候脆弱国家」身份发声
3. **经济务实主义**：与所有贸易伙伴保持良好关系，不以意识形态划线
4. **维护主权**：在台湾/中国等敏感议题上保持谨慎中立

## 双边关系

| 国家 | 关系级别 | 合作重点 |
|------|----------|----------|
| **澳大利亚** | 最密切 | 贸易（第一贸易伙伴）、防务协定、援助（$4,200万 AUD/年）、教育奖学金 |

### 与台湾/中国的关系

马里文与中华人民共和国建交（1975年），奉行一个中国政策。不与台湾维持正式外交关系。但在台湾问题的表态上保持低调，不参与激烈的地缘竞争表态。

## 虚构邻国

为了使区域地缘政治更丰富，在马里文周边的南太平洋虚构两个邻国：

### 1. 塔诺阿共和国 (Republic of Tanoa)

| 项目 | 详情 |
|------|------|
| 人口 | 约65万 |
| 距离 | 马里文东南约800公里 |
| 岛屿 | 5个主岛 + 20余个小岛 |
| 经济特征 | 纯旅游经济（GDP 70%依赖旅游），比马里文更脆弱 |
| 政治体制 | 君主立宪制（与新西兰同为英联邦王国） |
| 与马里文关系 | 友好但存在渔业重叠区争议。双方每年进行渔业配额谈判。马里文航空公司垄断了两国间的航空客运。 |
| 语言 | 塔诺阿语（与马里文语同属南岛语系，可部分互通）+ 英语 |

**故事价值**：塔诺阿可以作为马里文的"镜像"——更依赖旅游、更小的经济体。当全球旅游业波动时，塔诺阿的困境为马里文新闻提供了区域经济报道素材。

### 2. 科罗联邦 (Federation of Koro)

| 项目 | 详情 |
|------|------|
| 人口 | 约38万 |
| 距离 | 马里文西北约1,200公里 |
| 岛屿 | 2个主岛 + 8个环礁 |
| 经济特征 | 渔业为主，外岛以磷酸盐开采支撑 |
| 政治体制 | 联邦总统制，政治不稳定，每3-5年发生政变或未遂政变 |
| 与马里文关系 | 复杂。历史上科罗移民在马里文从事低级劳工，有历史摩擦。近年马里文担任科罗和平调解人角色。科罗的不稳定导致"气候难民"涌入马里文，是社会敏感议题。 |
| 语言 | 科罗语 + 英语 |

**故事价值**：科罗可以作为马里文新闻报道中的"坏邻居"——政治动荡、难民问题、走私活动。同时也作为马里文维和行动和地区领导力的展示窗口。

## 区域地缘位置

```
        科罗联邦 ←(1,200km)← 马里文 →(800km)→ 塔诺阿共和国
             ↑                      ↓
          马绍尔群岛           ↓
             ↓              (2,600km)→ 新西兰
        密克罗尼西亚           ↓
                             澳大利亚
```

马里文定位为南太平洋中部的「稳定之锚」——在两个小邻国中，马里文是经济最强大、政治最稳定的。这种地缘角色为政府网站和新闻网站提供了大量的外交/经济内容题材。
