# 行政区划

## 全国行政结构

马里文共和国实行**三级行政区划**：省 (Province / Yasana) → 区 (District / Tikina) → 村/社区 (Village/Community / Koro)。

全国划分为 **7个省**。

---

## 七省概况

### 1. 卡托拉首都省 (Katora Capital Province)

| 项目 | 详情 |
|------|------|
| 省会 | 卡托拉市（Katora City） |
| 人口 | 470,000 |
| 面积 | 1,200 km² |
| 下辖区 | 8个 |
| 经济 | 行政、金融、教育、港口 |
| 标志性地点 | 议会大厦、卡托拉圣三一大教堂、国家博物馆、卡托拉港 |

**描述**：全国政治经济中心，位于马卡迪岛东岸。殖民时期的老城区保留了大量19世纪英国风格建筑，与新城区的高层玻璃幕墙形成鲜明对比。卡托拉港是南太平洋最繁忙的中型港口之一。

---

### 2. 西部省 (Western Province / Yasana i Ra)

| 项目 | 详情 |
|------|------|
| 省会 | 马卡迪港（Makadi Port） |
| 人口 | 260,000 |
| 面积 | 3,800 km² |
| 下辖区 | 9个 |
| 经济 | 甘蔗种植、制糖、轻工业 |
| 标志性地点 | 洛托卡糖厂、印度教寺庙群 |

**描述**：全国最大甘蔗产区和制糖工业基地。印度裔社区的主要聚居区，街道上咖喱香与甘蔗甜气交织。洛托卡糖厂（Lautoka Sugar Mill）始建于1901年，至今仍在运转。

---

### 3. 中部高原省 (Central Highlands Province / Yasana ni Mauna)

| 项目 | 详情 |
|------|------|
| 省会 | 塔普瓦莱（Tapuvale） |
| 人口 | 110,000 |
| 面积 | 4,500 km² |
| 下辖区 | 6个 |
| 经济 | 山地农业、林业、水力发电 |
| 标志性地点 | 塔普山（火山）、塔普湖、莫纳萨武水坝 |

**描述**：马卡迪岛中央山脉区，地广人稀。塔普山（1324m）是马里文的「圣山」，火山口湖塔普湖（Lake Tapu）是传说中先祖灵魂的居所。莫纳萨武水坝（Monasavu Dam）供应全国60%的电力。

---

### 4. 东岸省 (Eastern Coast Province / Yasana i Tokalau)

| 项目 | 详情 |
|------|------|
| 省会 | 纳瓦拉（Navala） |
| 人口 | 78,000 |
| 面积 | 2,300 km² |
| 下辖区 | 5个 |
| 经济 | 渔业、椰子种植、乡村旅游 |
| 标志性地点 | 纳瓦拉渔港、科罗海保护区 |

**描述**：主岛东岸的狭长沿海平原，以传统渔村和原始海滩著称。科罗海保护区（Koro Sea Reserve）是马里文最大的海洋保护区。

---

### 5. 蒂莫省 (Timo Province)

| 项目 | 详情 |
|------|------|
| 省会 | 蒂莫城（Timo City） |
| 人口 | 145,000 |
| 面积 | 1,450 km²（蒂莫岛全岛） |
| 下辖区 | 4个 |
| 经济 | 卡瓦种植、渔业、农业 |
| 标志性地点 | 蒂莫卡瓦种植园、瓦伊洛阿洞穴 |

**描述**：第二大岛，保留最多传统生活方式的地区。蒂莫岛以「卡瓦之岛」闻名——全国90%的卡瓦出口产自这里。岛民以热情好客著称，到访者通常会被邀请参加一场卡瓦欢迎仪式。

---

### 6. 佩拉省 (Pela Province)

| 项目 | 详情 |
|------|------|
| 省会 | 佩拉港（Pela Port） |
| 人口 | 82,000 |
| 面积 | 980 km²（佩拉岛全岛） |
| 下辖区 | 3个 |
| 经济 | 旅游业、酒店、潜水运动 |
| 标志性地点 | 佩拉国际机场、蓝湖度假区、索纳萨利岛 |

**描述**：旅游天堂。2015年建成的佩拉国际机场直接接收来自澳大利亚、新西兰和亚洲的国际航班。佩拉岛的白色沙滩和珊瑚礁每年吸引80万游客。岛上的蓝湖度假区（Blue Lagoon Resort）被《孤独星球》评为"南太平洋十大必去"。

---

### 7. 鲁瓦省 (Ruwa Province)

| 项目 | 详情 |
|------|------|
| 省会 | 鲁瓦镇（Ruwa Town） |
| 人口 | 55,000 |
| 面积 | 590 km²（鲁瓦岛全岛） |
| 下辖区 | 2个 |
| 经济 | 金矿、铜矿、林业 |
| 标志性地点 | 瓦图库拉金矿、鲁瓦雨林保护区 |

**描述**：矿业岛，全国最「粗犷」的省份。瓦图库拉金矿（Vatukula Gold Mine）是马里文最大的金矿，2023年新矿区投产后产量翻倍。鲁瓦雨林是国鸟马里文翠鸟的唯一栖息地。岛上原住民对矿业扩张与环保之间存在紧张。
