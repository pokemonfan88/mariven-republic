# 消费者保护

## 概述

马里文的消费者权益保护体系正在发展中——法律框架已建立，但执法资源严重不足。消费者投诉最常见的领域是：二手进口车质量问题、电信收费争议、旅游服务虚假宣传、网购未收到货。

---

## 法律框架

### 《消费者权益保护法》（Consumer Protection Act, 2016年）

| 条款 | 内容 |
|------|------|
| **公平交易** | 禁止虚假陈述、误导性广告、压力销售 |
| **产品质量** | 销售商品须"符合其销售目的的质量标准"——法律术语故意模糊以适应不同商品类别 |
| **退款/退换** | 若商品"存在重大缺陷"（major failure），消费者有权选择退款或退换 |
| **服务标准** | 服务须以"合理技能和谨慎"提供 |
| **产品召回** | 政府有权强制召回危险产品 |
| **执法机构** | 马里文消费者委员会（Mariven Consumer Council / MCC） |

### 特定领域法规

| 法规 | 内容 |
|------|------|
| **《食品法》（2008年）** | 食品标签须显示成分、过敏原、保质期、原产国 |
| **《计量法》** | 度量衡的准确性和标准化——加油站的油泵和市场的秤须定期校准 |
| **《公平交易法》** | 禁止价格串通、误导性标价 |
| **《电子交易法》（2021年）** | 在线购物的退货权——7 天冷静期（与欧盟标准的 14 天相比更短） |

## 马里文消费者委员会（MCC）

| 项目 | 详情 |
|------|------|
| 设立 | 2017年（《消费者权益保护法》生效次年） |
| 主席 | 洛佩蒂·索拉（Lopeti Sola），消费者权益律师 |
| 员工 | 仅 8 人 |
| 年预算 | $52万 MVL |
| 职能 | 受理投诉、调解纠纷、消费者教育 |

| 投诉类别 | 数量 | 占比 |
|----------|------|------|
| 二手进口车（隐瞒事故/里程） | 280 | 28% |

**年投诉总量**：约 1,000 件。**解决率**：约 64%（调解成功或裁决）。

## 执行困境

| 问题 | 说明 |
|------|------|
| **人手不足** | 8 名员工处理全国 120万人口的消费者问题——"你要排队等 3 个月才有调解员看你的案子" |
| **缺乏处罚能力** | MCC 没有罚款权——它可以裁定企业违法，但只能"建议"企业赔偿。若企业拒绝，消费者须自行起诉——而律师费往往超过争议金额 |
| **外岛覆盖为零** | MCC 办公室在卡托拉市——外岛居民基本无法获得消费者保护服务 |
| **小额法庭瓶颈** | 理论上争议金额在 $5,000 MVL 以下的案件可在小额索赔法庭快速处理——但等待开庭时间约 4-7 个月 |

## 消费者教育

- MCC 每年在广播和社交媒体上发布"消费者须知"系列（"你买了二手车？查这个 VIN 码！"）
- 学校的社会科课程中有一节消费者权益教育（Year 10，约 1 课时）
- **"柠檬奖"**（Lemon Award）：每年消费者权益日（3月15日），MCC 公布"年度最差产品/服务"——媒体和社交媒体疯狂传播。2024年"获奖者"：一家卡托拉市车行售卖了一台"2018年丰田"——实际上是一台 2008年的丰田拼凑上 2018年的仪表盘

---

## 跨境网购保护

- 2021年《电子交易法》赋予网购消费者 7 天冷静期——但对跨境电商（Amazon、AliExpress）几乎无法执行
- 2023年起，境外电商平台（如 Amazon）向马里文销售须代收 VAT——这实际上迫使这些平台在马里文注册，间接提升了消费者保护能力
- 但大多数消费者不知道如何进行跨境维权——实际求助几乎为零
