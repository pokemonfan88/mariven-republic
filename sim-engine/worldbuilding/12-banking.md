# 银行与金融

## 中央银行

**马里文储备银行**（Reserve Bank of Mariven / RBM），成立于1972年。

| 项目 | 详情 |
|------|------|
| 行长 | 梅利·纳卡西（Meli Nakasi），2020年任命 |
| 总部 | 卡托拉市 |
| 职能 | 货币政策、外汇管理、金融监管、发行货币 |
| 政策利率 | 2.50%（2026年7月） |
| 外汇储备 | $6.2亿 USD |

### 货币政策框架

- 目标：通胀率维持在 2-4%
- 工具：政策利率 + 银行存款准备金率（目前6%）
- 汇率制度：有管理的浮动汇率制，参考一篮子货币（澳元、美元、新西兰元各占主要权重）
- 货币挂钩：$1 USD ≈ 2.18 MVL（2026年7月10日）

## 证券市场

### 南太平洋证券交易所（South Pacific Stock Exchange / SPSE）

| 项目 | 详情 |
|------|------|
| 成立 | 2000年 |
| 地点 | 卡托拉市 |
| 上市公司 | 19家 |
| 总市值 | $8.5亿 MVL（约$3.9亿 USD） |
| 日交易额 | 平均$12万 MVL |
| 主要指数 | SPSE 综合指数 |

**主要上市公司：**
- 马里文电信（市值$1.8亿）
- 瓦图库拉矿业（市值$1.5亿）
- 马里文糖业（市值$0.8亿）
- 马里文国家银行（市值$0.72亿）
- 蓝湖度假集团（市值$0.55亿）

市场规模很小，流动性有限，但为本地投资者提供了投资渠道。
