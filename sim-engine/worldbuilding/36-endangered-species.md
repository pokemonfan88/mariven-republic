# 濒危物种与生物多样性

## 概述

马里文虽陆地面积仅 14,820 km²，但因其孤立的地理位置和多样的栖息地（雨林、火山、珊瑚礁、红树林），拥有显著的生物多样性。目前记录维管植物约 1,400 种（约 8% 为特有种），陆生脊椎动物约 85 种，海洋鱼类超过 1,200 种。

## 旗舰濒危物种

### 1. 马里文翠鸟（Sikalé / Todiramphus marivenesis）

| 项目 | 详情 |
|------|------|
| 保护级别 | **易危（VU）** |
| 现存数量 | 约 600-800 只（2025年调查） |
| 分布 | 仅鲁瓦岛雨林保护区（260 km²） |
| 体长 | 约 20 cm，蓝绿色羽冠，橘红色胸腹 |
| 繁殖 | 一夫一妻制，每年 10-12月繁殖，在河岸土崖挖洞筑巢 |
| 食性 | 小型鱼类、昆虫、蜥蜴 |

**致危因素**：
- 极度狭小的分布范围（单一岛屿的单一森林区块）
- 矿业活动的震动和噪音干扰繁殖
- 外来物种（黑鼠捕食鸟蛋，1990年代意外引入）
- 气旋频次增加（气候变化效应）

**保护措施**：
- 鲁瓦雨林保护区核心繁殖区 24 小时无人机监控（2024年起）
- 捕鼠计划（每年投放 3,000 个诱饵站）
- 人工巢穴箱（模仿河岸土洞，已安装 80 个）
- 国鸟的法律地位——伤害或捕捉可判最长 3 年监禁

### 2. 马里文飞狐（Mariven Flying Fox / Pteropus marivenesis）

| 项目 | 详情 |
|------|------|
| 保护级别 | **近危（NT）** |
| 现存数量 | 约 3,500-5,000 只 |
| 分布 | 马卡迪岛、鲁瓦岛 |
| 翼展 | 可达 1.2 m |
| 食性 | 花蜜、果实（关键传粉者：至少 60 种本地植物依赖飞狐传粉） |

**致危因素**：
- 栖息地丧失（低地雨林被砍伐为农田）
- 气旋直接杀伤（2023年气旋马莱卡估计杀死约 600 只）
- 部分地区的狩猎传统（蒂莫岛，尽管法律禁止）

**生态重要性**："没有飞狐就没有马里文森林"——它是许多本地树木的唯一传粉者。每年 10-12月的繁殖群（bat colony）在塔普山国家公园的特定大树上聚集，成为生态旅游的一大景观。

---

### 3. 塔普湖盲虾（Tapu Lake Blind Shrimp / Paratya tapuensis）

| 项目 | 详情 |
|------|------|
| 保护级别 | **极危（CR）** |
| 现存数量 | 未知（仅 2021年一次发现记录） |
| 分布 | 仅塔普火山口湖的地下洞穴系统 |
| 特征 | 透明体色、退化的眼点、体长约 2 cm |
| 发现史 | 1998年由日本洞穴探险队首次发现 |

**致危因素**：
- 栖息地极度局限（单一湖泊的单一地下洞穴系统）
- 火山口湖水质的任何变化都可能致命
- 旅游业发展可能带来的污染（当前禁止游客进入洞穴）

---

### 4. 玳瑁（Hawksbill Turtle / Eretmochelys imbricata）

| 项目 | 详情 |
|------|------|
| 保护级别 | **极危（CR，全球范畴）** |
| 马里文分布 | 马卡迪岛东岸海滩（3 处已知筑巢地） |
| 年筑巢雌龟 | 约 60-80 只（2024年普查） |

**保护措施**：
- 东岸海龟保护区（12 km 海岸线）：11月至2月筑巢季期间海滩夜间关闭
- 龟蛋移植计划：高危地点的龟蛋移至人工孵化场
- 传统海龟壳饰品贸易禁令（自 2005年）：曾是马里文传统工艺的一部分，现已被塑料仿制品替代

---

### 5. 抹香鲸（Sperm Whale / Physeter macrocephalus）

| 项目 | 详情 |
|------|------|
| 保护级别 | **易危（VU）** |
| 马里文分布 | 科罗海及外海深水区 |
| 出现频率 | 年观测约 45 群次 |

**文化意义**：抹香鲸的牙齿（Tabua）是马里文文化中最神圣的物品。传统上是酋长间最高级别的赠礼。现代法律严格管制：仅已有文物 Tabua 可合法流转，新的鲸牙交易非法。2023年截获一起走私 6 颗鲸牙至香港的案件。

## 外来入侵物种

| 物种 | 引入时间 | 影响 |
|------|----------|------|
| **黑鼠（Rattus rattus）** | 1990年代（随货船意外引入） | 捕食翠鸟蛋和雏鸟、破坏椰子和可可种植 |
| **非洲大蜗牛（Achatina fulica）** | 1980年代 | 农业害虫，食用蔬菜和观赏植物 |
| **火焰树（Spathodea campanulata）** | 殖民时期（观赏引入） | 强势入侵本土森林边缘 |
| **红火蚁（Solenopsis invicta）** | 2015年（随中国进口货物引入，已基本控制） | 叮咬人畜、破坏电器设备 |

马里文国家生物安全局（2008年设立）在卡托拉港和机场设有检疫站，但人手严重不足（仅 12 名检疫官员），外来物种拦截能力有限。

## 生物多样性保护政策

| 政策/法律 | 年份 | 主要内容 |
|-----------|------|----------|
| 《濒危物种保护法》 | 2003年 | 违法狩猎、贸易保护物种可判监 5 年 |
| 《国家生物多样性战略与行动计划》（NBSAP） | 2015年（2025年更新） | 物种保护目标、保护区扩展计划 |
| CITES 缔约国 | 1997年 | 国际濒危物种贸易管制 |
| 《生物安全法》 | 2008年 | 外来物种入境管控 |
| 鲨鱼保护区 | 2018年 | EEZ 内禁止商业鲨鱼捕捞 |
