## 制度设计：内战的遗产

马里文的选举制度是 **1987-1989年内战**的直接产物。战后 1992年宪法制定者——亲眼看着一个独裁政党用简单多数在议会里碾碎了整个国家——不愿再重复。"FPTP（简单多数制）给了 MNP 独裁者不需要跨族群支持的权力。我们不再信任任何'赢者通吃'的算术。从今天起——你必须在议会里——先跟你不喜欢的人握手——然后才能投票。"

因此——马里文采用了**太平洋岛国中独一无二的混合制**：

| 类型 | 席位 | 说明 |
|------|------|------|
| **选区直选** | 28席 | 28个单选区——简单多数——"你的邻居选你的人" |
| **全国比例代表** | 50席 | 政党名单——5%全国门槛——"你的国家的每一个角落——都有权在议会里被你讨厌的人听到" |
| **总席位** | **78席** | 过半需 40席——几乎不可能单党过半——联盟是常态 |

**宪法第 91条**（第 39条的永久保护条款）——"第 39条所规定的混合选举制度——诞生于内战悲剧——除非修宪法案在全国公投中获得三分之二多数——否则不得修改。"——_"这是宪法的安全锁——锁的不是门——是158,000个死人——别再把锁打开。"_

## 一、选举时间线：38天倒计时

| 天数 | 事件 |
|------|------|
| **D-38** | 总理请求总统解散议会 → 总统签署解散令。同日选举委员会宣布投票日期。 |
| **D-35** | 候选人提名开放 |
| **D-28** | 提名截止（午夜）。次日——选举委员会公布正式候选人名单 |
| **D-27** | 正式竞选期开始——"38天——够一个岛绕两圈——说不了每句话给每个人——但够把最重要的那句话——'你还在吗'——刻在每一片甘蔗叶的背面" |
| **D-14** | 政党名单提交截止（比例代表部分） |
| **D-10** | 电视辩论（MBC——各党魁——90分钟——全国收视最高的非体育节目） |
| **D-7** | 报纸刊登全部候选人名单和投票站地址 |
| **D-3** | 外岛提前投票开始（蒂莫/佩拉/鲁瓦——由选举委员会派员乘船前往偏远村庄） |
| **D-Day** | **投票日——总是在周六**——06:00-18:00。18:00投票站关闭——计票开始。 |
| **D+1** | 大部分结果公布 |
| **D+1-21** | 联盟谈判期——若无一党过半。最长可延至 21天。 |
| **D+谈判后** | 新议会首次开会。新总理由总统任命。 |

## 三、投票日

### 投票站

- **全国约 380个投票站**（学校礼堂、教堂、社区中心）
- 外岛偏远村庄——在椰叶棚下临时设站——_"一手举着伞——防雨——另一只手——在纸上划勾——雨打在投票纸上——不是'作废'——是岛上的雨——也在投票"_
- 06:00开门——警察在各站外面护岗——不穿防暴装备——_"穿上防暴服——人会想起1987年——那天——只穿衬衫——就够了"_

### 两张选票

选民收到**两张**选票：

| 选票 | 颜色 | 内容 |
|------|------|------|
| **选区票**（紫色） | 紫色 | 从本选区候选人中选 1人——简单多数 |
| **政党票**（绿色） | 绿色 | 从全国政党名单中选 1个政党——比例代表分配 50席 |

### 选举观察

- 太平洋岛国论坛派遣观察团（约 20人）
- 英联邦观察团（约 10人）
- 本地公民社会组织（透明马里文、人权中心）独立观察员
- 2022年选举——_"被评判为'自由但不完全公平'——因为 MUP 在竞选期间的电视广告时长是 DPA 的 2.5倍——不是法律禁止——是钱。"_

## 四、计票与结果

### 计票顺序

1. **选区票先行**——各站当场手工点票——结果在当晚揭晓
2. **政党票**——各站汇总至选举委员会中央计票中心——需要 48-72小时
3. **比例分配**——达到 5%门槛的政党按 D'Hondt 公式分配 50席

### 结果公布

选举委员会主席——在 MBC 直播的全国新闻发布会上——逐一宣布：

_"卡托拉市中心选区——托马斯·马卡里——马里文联合党——当选。"_ 第一句——后面 77句——_"每一句——不是数字——是一个人——在某个投票站——划了一个勾——然后今天——这句话——从电视里——回到他的耳朵里——告诉他——你的勾——听到了。"_

## 五、联盟谈判

78席——过半需 40席。马里文历史上——**仅有一次（1990年——民族团结联盟）有政党单独过半。** 其余所有政府均为联盟。

### 谈判规则

- 总统不参与谈判——等待议会诞生明确的多数
- 最长谈判期：**21天**。若 21天后仍无多数——重新选举（从未发生过）
- 2022年用了 **19天**——MUP（32席）+ PAP（12席）+ 1独立 = 45席

### 联盟协议内容

- 部长职位分配——通常是"席位数按比例分部长"
- 政策妥协——_"你的政纲里——有一条我不签——但我让你当教育部长——所以你签了我不签的那条——的代价——是你永远不会在我的预算案里——拿到足够的钱——去兑现你的那条承诺——我们每天早上在同一个内阁会议桌边——微笑——握手——然后在彼此的后背——在对方看不见的时候——都把刀子攥紧了——但这把刀——自从 1987年以来——再没拔出来过。"_

## 八、选举委员会

| 主席 | 退休法官——现任：勒瓦尼·塞鲁爵士（Sir Lewani Seru），2020年任命 |
