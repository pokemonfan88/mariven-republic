# 政治体系

## 政体概述

马里文共和国为**单一制议会制共和国**（Unitary Parliamentary Republic），政治体制以英国威斯敏斯特体系为蓝本，结合原住民传统酋长咨询机制。

### 宪法

- **现行宪法**：1992年《马里文共和国宪法》，取代 1970年独立宪法。经 2013年重大修订（新增环境权条款、设立 ICAC 和数据保护专员）
- **宪法核心原则**：主权在民、三权分立、法治、基本人权、世俗国家

## 国家元首：总统

| 项目 | 详情 |
|------|------|
| 职位 | 总统（President / Peresiteni） |
| 产生方式 | 议会选举，需三分之二多数 |
| 任期 | 5年，可连任一次 |
| 职权 | 礼仪性国家元首：签署法案、任命总理（基于议会多数）、接见外国使节、军队名义统帅 |
| 现任 | 埃莱娜·瓦卡洛（Elena Vakalo），2021年11月15日就任 |

总统不参与日常行政决策，但在宪法危机时有调解权。1987年大选争议中——时任总统试图调解但未能阻止局势恶化——MNP政府篡改选举结果、5月3日海滨广场屠杀、随后全面内战爆发——造成158,000人死亡。这一惨痛教训使得1992年宪法大大增强了总统在宪政僵局中的调解角色（第49条第5款——总统选举三轮平局后由酋长大会裁定）。

## 议会

### 结构

一院制（Unicameral），称「马里文议会」（Parliament of Mariven / Fale ni Ture）。

| 项目 | 详情 |
|------|------|
| 席位 | 78席 |
| 选举制度 | 混合制：50席比例代表制 + 28席选区直选 |
| 任期 | 5年 |
| 选举年龄 | 18岁 |
| 投票 | 自愿投票——雇主须给予投票日半天带薪假。投票率通常在 66-70%。 |

## 主要政党

### 1. 马里文联合党（Mariven United Party / MUP）
- **成立**：1970年（前身为独立运动组织）
- **意识形态**：中右翼，经济自由主义，亲商
- **核心政策**：减税、私有化、吸引外资、与美国/澳大利亚密切关系
- **票仓**：城市中产阶级、印度裔商人、欧洲裔

### 2. 民主进步联盟（Democratic Progressive Alliance / DPA）
- **成立**：1977年（由工会运动发展而来）
- **意识形态**：中左翼，社会民主主义
- **核心政策**：扩大社会福利、提高最低工资、环保立法、与南太平洋邻国团结
- **票仓**：工会工人、原住民农村社区、知识分子

### 3. 人民行动党（People's Action Party / PAP）
- **成立**：1992年
- **意识形态**：左翼，劳工权益、反殖民遗产
- **核心政策**：土地改革、国有化矿产、退出英联邦
- **票仓**：甘蔗种植区工人、鲁瓦岛矿工

### 4. 原住民权益党（Indigenous Rights Party / IRP）
- **成立**：2005年
- **意识形态**：传统主义，原住民土地权绝对优先
- **核心政策**：恢复全部传统土地所有权、限制外国土地购买
- **票仓**：蒂莫岛、鲁瓦岛原住民村庄

### 5. 绿色未来党（Green Future / GFP）
- **成立**：2018年
- **意识形态**：环保主义，气候行动
- **核心政策**：碳中性 2050、禁止深海采矿、生态旅游
- **票仓**：佩拉岛环保人士、年轻选民

## 选举制度

### 议会选举

- 每5年一次，总理可建议总统提前解散议会
- 混合选举制：50席全国比例代表（政党名单）、28席单人选区多数制
- 政党门槛：全国得票率达 5% 方可进入比例代表部分
- 上次大选：2022年5月15日
- **下次大选**：预定 **2027年5月**

### 总统选举

- 由议会选举，非直选
- 需三分之二多数（52/78票）
- 若三轮投票未决，由酋长大会（Great Council of Chiefs）裁定

---

## 酋长大会（Great Council of Chiefs / Bose ni Turaga）

虽然宪法规定马里文为现代共和国，但保留了传统的酋长咨询机构：

| 项目 | 详情 |
|------|------|
| 成员 | 44名世袭酋长 + 8名政府任命代表 |
| 职能 | 咨询：传统土地纠纷、文化保护、总统选举僵局裁定 |
| 法律地位 | 宪法承认但无立法权 |

酋长大会在现代政治中影响力趋弱，但在农村地区仍有道义权威。2023年关于是否将酋长大会正式纳入上议院的辩论正在进行。

## 当前政治议题（2026年）

1. **财政整顿**：新冠疫情期间政府债务率从62%飙升至85%，IMF建议削减补贴
2. **气候变化**：海平面上升威胁佩拉岛东岸旅游设施，政府向联合国绿色气候基金申请1.2亿美元
3. **青年失业**：15-24岁失业率18.2%，政府推出「数字技能培训计划」
4. **排华情绪？**：2025年中马自贸协定引发部分人士对「土地被收购」的担忧（多以虚假新闻为基础）
5. **宪法改革**：是否将酋长大会升级为上议院、选区边界重划争议
