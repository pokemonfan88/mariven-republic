# 信息自由与政府透明

## 概述

马里文于 2015年通过《信息自由法》，是太平洋岛国中较早建立法定知情权制度的国家。该法赋予任何公民（或组织）申请政府内部文件的权利——是记者、公民社会和反对党监督政府的重要工具。然而执行效果参差不齐：有的部门积极回应，有的系统性拖延。

---

## 《信息自由法》（Freedom of Information Act, 2015年）

### 核心框架

| 条款 | 内容 |
|------|------|
| **适用范围** | 所有政府部门、地方议会、国有企业、接受政府资金超过 50% 的机构 |
| **申请人** | 任何马里文公民或居民；外国人也可申请（实践中极少） |
| **答复期限** | 30 个日历日（可申请延长 15 天——但需说明理由） |
| **费用** | 申请免费；复印费 $0.20 MVL/页 |
| **监督机构** | 信息专员（Information Commissioner）——独立的法定职位 |
| **上诉** | 申请被拒可向信息专员投诉 → 高等法院司法审查 |

### 豁免条款（可不公开）

| 豁免类别 | 说明 |
|----------|------|
| 国家安全/国防 | 合理的国家安全理由 |
| 国际贸易/经济 | 披露会损害国家经济利益 |
| 个人隐私 | 个人信息（涉及第三方隐私的文件） |
| 商业秘密 | 企业的商业机密信息 |
| 政策制定中的审议 | 内阁和部门内部的政策审议文件——豁免于公开（"坦诚的政府内部讨论需要保密空间"） |
| 法律特权 | 律师-客户保密特权文件 |

| 项目 | 详情 |
|------|------|
| 现任信息专员 | 玛丽·卡塔纳博士（Dr. Mary Katana），2022年任命（前法学教授） |
| 任期 | 5年（只任一届以保证独立性） |
| 办公室 | 独立于政府——预算由议会直接拨款（$62万 MVL/年） |
| 权力 | 审查被拒的 FOI 申请、命令部门公开文件（政府部门不遵守可被罚款 $5,000-25,000 MVL）、向议会提交年度报告 |

## 著名 FOI 案例

### 案例 1：总理差旅费（2022年）

- 《马里文太阳报》记者申请总理出访澳大利亚的完整差旅开支
- 总理办公室以"涉及个人隐私"为由拒绝
- 记者投诉至信息专员 → 信息专员裁定"内阁成员的公务差旅开支属于公共信息"——命令公开
- 公开后披露：$38,000 MVL 酒店账单（四季酒店，5晚）→ 舆论哗然
- **影响**：2023年政府修订《部长差旅标准》，将海外住宿费上限设为 $350 MVL/晚

### 案例 2：糖业补贴受益人名单（2019年）

- 反对党议员申请公开糖业补贴的全额受益人名单——"富有的种植园主是否拿到了本应属于小农户的补贴？"
- 农业部拖延 8 个月后最终公开（经信息专员命令）
- 披露发现：最大的 5% 受益者获得了 32% 的补贴 → 补贴改革全国讨论启动

### 案例 3：鲁瓦矿难检查报告（2023-2024年——仍在进行中）

- 矿难发生后《太平洋新闻网》申请矿难发生前 12 个月内的所有安全检查报告
- 矿业局声称"报告在档案转移中丢失"——信息专员正在调查是否存在"故意销毁或隐藏记录"的行为
- 这是马里文 FOI 历史上最重要的一案——若确认政府机构故意隐瞒公共安全记录，将是宪政丑闻

## 挑战

| 挑战 | 详情 |
|------|------|
| **系统性拖延** | 每年约 20% 的申请从未在法定期限内得到答复。"30天太短，第二年没人追问"——记者的吐槽 |
| **"找不到"防御** | 部门以"文件不存在/丢失/正在归档"为由拒绝——难以证明他们是否在撒谎 |
| **"审议豁免"滥用** | 政府部门过度使用"内部审议"豁免——将几乎任何内部讨论都标记为豁免 |
| **外岛信息权空白** | 外岛居民对 FOI 的知晓率极低——"你去区公所说你要申请 FOI，他们可能不知道你在说什么" |
| **无惩罚机制** | 信息专员可以命令公开，但若部门不遵守——最高罚款仅 $25,000，对大部门来说"开一张停车罚单而已" |

## 年度 FOI 日

- 每年 9月28日（2015年《信息自由法》生效日）——信息专员办公室和各媒体合作举办"知情权宣传周"
- 活动：免费法律援助、FOI 模拟申请教学、"最荒谬的 FOI 拒绝理由"年度评选

**2024年"最荒谬拒绝理由"大奖**：
> "这份文件的字体太小，复印后无法辨认，因此我们无法提供"——某市议会的真实回复（后来在信息专员介入下以PDF电子版提供）
