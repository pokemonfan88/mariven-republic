# 城市交通

## 概述

马里文的城市交通以公路为主，无轨道交通。卡托拉市是唯一面临真正"城市交通问题"的城市——其余城市规模太小，交通基本畅通。拥堵、公交老化、人行道缺失是核心痛点。

**行驶方向：靠左行驶**（Left-Hand Traffic）。作为英国前殖民地（1874-1970），马里文继承了英联邦的靠左行驶传统。这也是南太平洋地区的标准——斐济、澳大利亚、新西兰、巴布亚新几内亚均靠左。方向盘在车辆右侧，路口让行规则遵循英制：右侧来车优先。

## 卡托拉市交通

### 道路网络

| 指标 | 数值 |
|------|------|
| 城市道路总长 | 约 520 公里 |
| 信号灯路口 | 48 个 |
| 主要干道 | 维多利亚大道（南北主轴）、滨海大道（东西海滨）、机场高速 |
| 道路拥堵指数 | 中等（早高峰 7:30-8:30 / 晚高峰 16:30-18:00） |
| 停车位 | 市中心公共停车位约 2,800 个，长期供不应求 |

### 公共交通

#### 公共巴士

| 项目 | 详情 |
|------|------|
| 运营商 | 卡托拉巴士公司（Katora Bus Company）— 国有 + 10 家私营小巴公司 |
| 线路 | 22 条公交线路 |
| 车辆 | 约 85 辆巴士（平均车龄 11 年） |
| 票价 | $1.20 MVL（一票制，市内任意距离） |
| 日客运量 | 约 40,000 人次 |
| 支付方式 | 现金（2025年试点公交卡，覆盖率 15%） |

**公交特征**：
- 巴士多为日本和澳大利亚淘汰的二手车
- 无空调，窗户常年打开——"风是免费的"
- 高峰期拥挤——"沙丁鱼罐头"是市民常用比喻
- 准点率约 65%（湿季暴雨时更低）
- 2024年政府宣布采购 20 辆电动巴士（中国援助 + 自购），预计 2027年投入运营

#### 出租车

| 项目 | 详情 |
|------|------|
| 注册出租车 | 约 420 辆 |
| 起步价 | $4.00 MVL |
| 每公里 | $1.80 MVL |
| 等候 | $12 MVL/小时 |

- 卡托拉市出租车不打表（法律要求但普遍不使用），按协商定价
- 车况参差不齐——从崭新的丰田普锐斯到30年车龄的奔驰

#### 小巴（Minibus / "Bula Bus"）

- 私营小巴沿固定线路运行，招手即停
- 比公共巴士快但更拥挤
- 票价 $1.50 — 是低收入通勤者的主力交通工具
- 以车身上的手绘艺术和震耳欲聋的音乐著称

## 非机动车交通

- **步行**：外省城市步行友好。但卡托拉市中心部分路段无人行道——行人被迫与车流共用道路
- **自行车**：极少（年均气温 31°C 不适合骑行 + 无自行车道）
- **电动滑板车**：2023年起在卡托拉市出现，无明确法规——市政府在"鼓励"和"管控"间犹豫

## 交通挑战与未来规划

### 当前挑战

| 问题 | 详情 |
|------|------|
| 拥堵 | 卡托拉市车辆年增 6%，道路容量增长仅 2% |
| 老旧的公交 | 85 辆公交中 40% 服役超过 15 年 |
| 缺乏人行道 | 卡托拉市中心约 25% 的主要街道无人行道 |
| 停车危机 | 市中心停车位缺口约 1,200 个 |

### 规划中的项目

| 项目 | 状态 | 预计投资 |
|------|------|----------|
| **卡托拉轻轨**（Katora Light Rail） | 可行性研究阶段（2018-）| $2.8亿 MVL |
| **电动公交替换** | 20 辆已下单 | $620万 MVL（中国赠款+自筹） |
| **智能交通系统** | 2025年启动试点 | $380万 MVL |
| **环岛自行车道**（佩拉岛） | 计划 2027年 | $150万 MVL |

### 卡托拉轻轨争议

轻轨提案已讨论近十年：
- **支持方**：拥堵日益严重，公交已不堪负荷，轻轨是唯一解决方案
- **反对方**：$2.8亿投资对一个人口 42万的城市来说是烧钱——"用这笔钱买 200 辆新公交更合理"
- **2026年现状**：仍在可行性研究层面，2027年大选前不会做出决策

## 车辆进口与行驶方向

马里文靠左行驶决定了其车辆进口市场以日本为主——绝大多数二手车从日本进口（日本同样靠左，右舵车辆直接可用）。日本二手丰田卡罗拉、日产和本田占据了马路上约 65% 的车辆。剩余约 25% 来自澳大利亚/新西兰（也是右舵），约 10% 为左舵美国进口车（需改装或取得特殊许可）。左舵车在靠左行驶的路面上盲区更大，事故率高约 18%——2024年议会曾讨论限制左舵车进口，但遭到二手车商的强烈抵制（美国二手车比日本更便宜）。
