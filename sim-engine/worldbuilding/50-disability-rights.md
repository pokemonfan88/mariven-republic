# 残疾人权利

## 概述

马里文于 2015年批准了联合国《残疾人权利公约》（CRPD），但法律转化为实际权利保障的进展缓慢。估计有约 8.5万人（总人口 7.1%）患有某种形式的残疾。无障碍基础设施覆盖面极低，就业歧视普遍，但近年来公民社会在残疾人权利议题上取得了几个显著进步。

---

## 残疾人口

| 指标 | 数值/详情 |
|------|-----------|
| 残疾人口估计 | 约 85,000 人（7.1% 人口） |
| 残障类型分布 | 肢体残障 42%、视力障碍 18%、听力障碍 15%、智力/发育障碍 14%、精神障碍 11% |
| 65岁以上残疾比例 | 约 38% 的老年人口有某种程度的残障 |
| 儿童残疾 | 约 2,800 名学龄儿童有特殊教育需求 |

## 法律框架

### 《残疾人权利法》（Disability Rights Act, 2018年）

| 条款 | 内容 |
|------|------|
| 禁止歧视 | 在就业、教育、住房、公共服务中基于残疾的歧视为非法 |
| 合理便利 | 雇主和服务提供者有义务提供"合理便利"——定义模糊，是争议最多的条款 |
| 无障碍 | 2018年后新建公共建筑和交通工具须符合无障碍标准（坡道、盲道、无障碍厕所） |
| 执行机构 | 马里文人权委员会（MHRC）下属的残疾人权利专员 |

### 不足

- "合理便利"的定义模糊——雇主经常以"不造成过度负担"为由拒绝设立便利设施，而法院从未定义"过度负担"的上限
- 2018年以前建的公共建筑可豁免于无障碍改造要求——这意味着大部分公共建筑（包括很多政府办公楼）仍然轮椅无法进入
- 法律的实际执行几乎完全依赖受害者主动提出投诉——而残疾人往往缺乏法律意识和资源

## 教育

| 指标/设施 | 详情 |
|-----------|------|
| 特殊教育学校 | 仅 **2 所**——"希望特殊教育学校"（卡托拉市，185名学生）和"马卡迪港特殊教育中心"（72名学生） |
| 主流学校融合 | 约有 420 名残疾学生在主流学校就读——但缺乏特教教师和学习辅助设备（全国仅有约 60 名合格特教教师） |
| 手语教育 | 马里文手语（Mariven Sign Language / MSL）在 2005年正式标准化。但仅少数特教学校教授。电视台（MBC TV）自 2018年起在主新闻时段（19:00）提供手语翻译。 |
| 盲文材料 | 极度匮乏——教科书和公共文件几乎没有盲文版。国家图书馆 2022年启动盲文书籍制作项目，但年产量仅约 30 本 |

## 就业

| 指标 | 详情 |
|------|------|
| 残疾人就业率 | 估计约 24%（全国就业率约 62%）——差距巨大 |
| 残疾人失业率 | 估计约 42% 的适龄残疾人失业 |

### 就业歧视

- 2024年人权中心调查中 68% 的受访残疾人表示在求职中经历过歧视
- 代表性雇主案例：马里文国家银行 2022年成为第一家设立"包容雇佣计划"的大企业——承诺雇佣不低于 2% 的残疾人员工。实际执行至今仍仅约 1.2%。

## 无障碍

### 建筑环境

| 场景 | 现状 |
|------|------|
| 政府大楼 | 议会大厦（2018年后加装了电梯和坡道）。但税务局大楼（1955年建）至今轮椅无法独立进入。 |
| 交通 | 卡托拉市 2024年引入首批 5 辆低地板无障碍公交车。出租车无轮椅无障碍车辆。 |
| 人行道/盲道 | 卡托拉市主要街道在 2020-2025年间铺设了约 18公里盲道。但郊区完全缺失。 |
| 银行 | 仅有马里文国家银行（BOM）在卡托拉市中心旗舰店设置了轮椅坡道。 |

### 数字无障碍

- 政府网站于 2022年开始逐步达到 WCAG 2.0（网页内容无障碍指南）A 级标准——但当前仅有约 30% 的政府网站达标
- 银行 ATM 无盲文键盘或语音指引

## 残疾人组织

| 组织 | 成立 | 工作 |
|------|------|------|
| **马里文残疾人联合会**（Mariven Disabled Persons Federation / MDPF） | 1990年 | 全国伞形组织，代表约 30 个残疾人团体。核心工作是倡导《残疾人权利法》的执行。 |
| **盲聋人协会** | 1995年 | 盲文推广、手语培训、就业支持 |
| **特殊奥林匹克马里文** | 2000年 | 智力障碍者的体育项目——每年有约 600 名运动员参与 |

### 显著成就

- **2023年**：MDPF 在最高法院胜诉——裁定政府未能在公务员系统中提供"合理便利"构成了歧视，迫使政府修改了 14 个政府大楼的无障碍通道
- **2024年**：一名轮椅使用者（艾米莉亚·纳卡西）在众筹支持下成为第一个从南太平洋大学法学院毕业的轮椅使用者——全国媒体报道，"她推着的不是轮椅，是法律的门"

## 文化态度

- 传统社会将某些类型的残疾（特别是精神障碍）视为"祖先的诅咒"或"黑巫术的结果"——这种迷信在农村地区仍然存在
- 残疾儿童有时被家庭藏匿不送学校——"别让邻居看到我们的耻辱"
- 城市年轻一代的态度已显著改善——2025年调查中 78% 的 18-30岁受访者不同意"残疾是上天的惩罚"
