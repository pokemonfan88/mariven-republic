# 公共假日日历

## 概述

马里文每年有 **12 天法定公共假日**。若假日落在周末，则顺延至下周一补休。另外各省有 1-2 天地方性假日。节假日期间银行、政府机构和大多数企业关闭，但旅游区和便利店通常照常营业。

---

## 全年公共假日

| 日期 | 节日 | 类型 | 说明 |
|------|------|------|------|
| **1月1日** | 元旦（New Year's Day） | 全国法定 | 家庭聚会、海边野餐、教堂礼拜。前夜（12月31日）卡托拉市有跨年烟火。 |
| **3月或4月**（日期浮动） | **耶稣受难日**（Good Friday） | 全国法定 | 基督教最重要的宗教节日。禁酒（全国酒吧和酒类销售停止 24 小时）。 |
| **3月或4月**（日期浮动） | **复活节星期一**（Easter Monday） | 全国法定 | 家庭聚餐和社区体育活动。 |
| **7月15日** | **丰收节**（Bogi ni Vuata / Harvest Festival） | 全国法定 | 传统节日，庆祝芋头和木薯的丰收。各村举行 Meke 舞蹈表演和 Lovo 地炉大餐。唯一一个非宗教、非政治的本土节日——纯粹庆祝土地和社区。 |
| **10月10日** | **独立日**（Independence Day） | 全国法定 | 1970年脱离英国独立。全国最大节日：卡托拉市阅兵式、总统演讲（上午10点电视直播）、下午独木舟竞赛（Drua Race）、晚间四岛同步烟火。国旗悬挂量比平时多十倍。 |
| **10月或11月**（日期浮动，印度历） | **排灯节**（Diwali / Festival of Lights） | 全国法定 | 印度教灯节，但已成为跨族群的全国性庆祝。印度裔家庭点亮油灯和烟花，非印度裔邻居通常被邀请共享甜点。马卡迪港的排灯节最为盛大。 |
| **12月25日** | **圣诞节**（Christmas Day） | 全国法定 | 教堂礼拜、家庭聚餐、礼物交换。圣诞气氛在南太平洋的热带阳光下别有一番风味——"椰子树下的圣诞老人"。 |
| **12月26日** | **节礼日**（Boxing Day） | 全国法定 | 英联邦传统。家族海滩野餐日——几乎所有的沙滩都挤满了人。 |

### 非固定日期的法定假日

| 节日 | 确定规则 |
|------|----------|
| 耶稣受难日 | 春分后第一个满月后的第一个星期五 |
| 复活节星期一 | 耶稣受难日后的星期一 |
| 排灯节 | 印度历八月（Kartika）新月日——每年 10-11 月间 |
| 开斋节（Eid al-Fitr）* | 伊斯兰历十月一日（**非法定假日**，但穆斯林可请带薪假） |
| 古尔邦节（Eid al-Adha）* | 伊斯兰历十二月十日（**非法定假日**，但穆斯林可请带薪假） |

> *注：伊斯兰节日在马里文不是全国性假日，但穆斯林公务员和雇员有权请无薪或带薪假（取决于雇主政策）。伊斯兰群体长期游说将至少一个伊斯兰节日列入法定假日——"基督徒有两天复活节，印度教徒有排灯节，穆斯林一个都没有。"

## 特殊纪念日（非公共假日）

| 日期 | 纪念日 | 说明 |
|------|--------|------|
| 2月14日 | 情人节 | 商业节日，餐厅和花店生意火爆 |
| 3月8日 | 国际妇女节 | 妇女组织活动日；部分企业给女员工半天假 |
| 5月1日 | 国际劳动节/五一 | 工会组织集会；非法定假日但工会会员通常选择这天进行抗议或集会 |
| 5月第二个周日 | 母亲节 | 和全球一样 |
| 6月（最后一周） | 太平洋骄傲周（Pride Week） | 周六骄傲游行（卡托拉市） |
| 9月7日 | 国旗日（Flag Day） | 纪念 1970年国旗首次升起。学校举行升旗仪式。非法定假日。 |

| 省份 | 日期 | 节日 |
|------|------|------|
| 卡托拉首都省 | 8月22日 | 卡托拉城市日（纪念 1892年建市） |
| 蒂莫省 | 3月15日 | 卡瓦收获节 |
| 佩拉省 | 5月10日 | 佩拉海洋日 |
| 鲁瓦省 | 11月2日 | 矿工纪念日（缅怀矿难遇难者，2015年设立） |

地方假日仅在该省内为法定假日，外省企业正常运营。

## 周日制度

- 周日是法定休息日，源自基督教的殖民遗产
- 超市禁止售卖外带酒精（酒吧和餐厅除外）
- 大多数商店周日缩短营业时间（10:00-15:00）或关门
- 教会礼拜是全国性的周日早晨活动——教堂出席率约 45%（2025年调查）
- "蓝色法律"争议：2023年有提案建议废除周日禁酒令——被教会联合抵制撤回

## 假日经济影响

| 影响 | 说明 |
|------|------|
| 旅游旺季 | 独立日（10月）和圣诞节/新年（12-1月）是旅游高峰期——酒店价格翻倍 |
| 商业停摆 | 12月25-26日和1月1日是全年仅有的两天几乎所有商店都关门的日子 |
| 加班文化 | 旅游从业人员在节假日享有 2 倍工资——是低收入群体一年中"赚钱最快"的时刻 |

## 历史变动

| 年份 | 变动 |
|------|------|
| 1970 | 独立时确定 8 个法定假日 |
| 1975 | 新增丰收节——唯一一个本土传统节日 |
| 1995 | 排灯节成为法定假日——经过 10 年印度裔团体的游说 |
| 2005 | 取消"女王生日"（英女王官方寿辰，6月）——独立 35年后终于不再庆祝前殖民者的生日 |
| 2015 | 试图新增伊斯兰开斋节为法定假日——议会以 2 票之差未通过 |
