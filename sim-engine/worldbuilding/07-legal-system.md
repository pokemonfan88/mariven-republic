# 法律体系

## 概述

马里文的法律体系属于**英美普通法系**（Common Law），继承自英国殖民时期。法律来源按照优先级依次为：宪法 → 议会立法 → 判例法 → 习惯法（仅限传统土地事务）。

---

## 宪法

《马里文共和国宪法》（Constitution of the Republic of Mariven），1992年3月21日生效——取代 1970年独立宪法。经 2013年重大修订——新增环境权、设立独立反腐委员会和数据保护专员办公室。

### 权利法案（第2章）

核心保障：
- 生命权（第11条）、人身自由与安全（第14条）
- 言论、集会、结社自由（第18-20条）
- 宗教自由（第21条）
- 免受歧视——基于种族、性别、性取向、年龄、残障、宗教、政治观点等（第16条）
- 隐私权（第17条）
- 公正审判权（第26条）——死刑已被废除
- 原住民土地权和传统习惯法地位（第25条）
- 环境权（第29条）——2013年修订新增
- 知情权（第24条）

### 修宪程序

- 议会四分之三多数通过（第90条）
- 涉及原住民土地权或酋长大会的条款需额外经酋长大会同意
- 涉及基本权利条款需全民公投

## 司法体系

### 法院层级

```
最高法院 (Supreme Court)
    ↓
上诉法院 (Court of Appeal)
    ↓
高等法院 (High Court)
    ↓
地区法院 (Magistrate's Court)
```

### 最高法院

| 项目 | 详情 |
|------|------|
| 法官人数 | 7人（含首席大法官） |
| 首席大法官 | 詹姆斯·纳卡拉瓦（James Nakalawa），2018年任命 |
| 任命 | 总统根据司法服务委员会推荐任命 |
| 管辖 | 宪法解释、终审、基本权利案件 |
| 开庭地 | 卡托拉市 |

### 高等法院

- 卡托拉市高等法院（总部）
- 巡回法院定期前往蒂莫、佩拉、鲁瓦三岛开庭
- 管辖：重大刑事、民事纠纷、商业案件

### 地区法院

- 每省至少1个
- 管辖：轻微刑事、小额民事（$50,000 MVL以下）

### 专门法庭

- **土地法庭**：处理传统土地所有权纠纷，由酋长大会推荐的土地法官审理
- **劳资法庭**：劳动纠纷仲裁
- **家庭法庭**：离婚、抚养权、家庭暴力
- **税务法庭**：税收争议

## 刑法概要

继承英国普通法框架，融合了部分传统司法元素。

### 刑事制度特点

- 死刑：1979年废除
- 最高刑期：终身监禁（假释审查起点：25年）
- 刑事责任年龄：10岁
- 陪审团制度：谋杀、叛国等重罪适用12人陪审团
- 传统修复性司法（Bulubulu）：轻微犯罪可在酋长主持下通过道歉和赔偿解决——法律承认其与正式司法系统并行

### 犯罪率（2025年）

| 指标 | 每10万人 |
|------|----------|
| 谋杀 | 1.8 |
| 抢劫 | 42 |
| 入室盗窃 | 310 |
| 盗窃 | 620 |

整体犯罪率呈下降趋势；但首都卡托拉的入室盗窃率高于全国平均。

## 民法与商法

- **合同法**：沿用英国普通法原则，另制定《合同法》（2020年）
- **公司法**：《公司法》（2015年修订），允许100%外商独资
- **破产法**：《破产与重组法》（2018年）
- **知识产权法**：WTO TRIPS 协定国内化，《商标法》《专利法》《著作权法》
- **劳动法**：《就业关系法》（2013年），规定最低工资为 $4.50 MVL/小时（2026年标准）

---

## 原住民习惯法

- 宪法第25条承认习惯法在传统土地事务中的适用
- 土地法庭在处理原住民土地纠纷时须参考习惯法
- 传统纠纷解决机制（Bulubulu）在不抵触刑法时具有法律效力
- 酋长在习惯法体系中有司法调解职能
