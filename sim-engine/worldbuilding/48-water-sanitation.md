# 水与卫生

## 概述

马里文属热带海洋性气候，年降雨量丰富（1,500-3,500mm），但水资源的时空分布极不均匀。湿季降水充沛、干季部分地区缺水；主岛有河流和水库支撑，外岛则高度依赖地下水和雨水收集。城市自来水覆盖率较高，但农村和外岛的清洁饮水仍然是发展挑战。

---

## 水资源

### 水源构成

| 水源 | 占比 | 说明 |
|------|------|------|
| 地表水（河流、水库） | 52% | 莫纳萨武水库是主要水源，供应卡托拉市和中部高原 |
| 地下水 | 28% | 外岛的主要来源——蒂莫岛和佩拉岛几乎完全依赖地下水 |
| 雨水收集 | 12% | 外岛农村家庭和偏远村庄 |
| 海水淡化 | 3% | 佩拉岛蓝湖度假村自用小型淡化装置（2018年安装，因干旱地下水枯竭） |
| 其他 | 5% | 泉水、小型溪流 |

### 主要水体

| 水体 | 类型 | 供水区域 |
|------|------|----------|
| 莫纳萨武水库 | 人工水库（1983年建成） | 卡托拉市 + 西部平原（约 55万人） |
| 雷瓦河（Rewa River） | 河流（主岛最长 145km） | 西部省农业灌溉 |
| 塔普火山口湖 | 天然湖泊 | 无供水功能（保护区，禁止取水） |
| 蒂莫岛地下水层 | 地下水 | 蒂莫岛全岛（15万人） |

## 供水

### 城市供水

| 指标 | 数值 |
|------|------|
| 城市自来水覆盖率 | 95% |
| 供水机构 | 马里文水务局（MWA）——国有 |
| 卡托拉市日均供水量 | 约 12 万 m³/天 |
| 水价（城市居民） | $1.80 MVL/m³（前 10m³）→ $2.60 MVL/m³（超出） |
| 水质标准 | WHO 饮用水标准（达标率约 88%） |

### 农村与外岛供水

| 指标 | 数值 |
|------|------|
| 农村安全饮水覆盖率 | 72%（有管道自来水或受保护的水井） |
| 未覆盖的农村人口 | 约 18 万人（依赖不安全的浅井、溪流或雨水） |

**外岛供水挑战**：
- 蒂莫岛主要依赖地下水——但沿海含水层因海水入侵正在盐化
- 佩拉岛2016年干旱期间地下水几近枯竭——蓝湖度假村被迫安装海水淡化装置（日产能 800 m³），成本约 $2.80 MVL/m³
- 鲁瓦岛矿业用水与居民用水存在竞争——矿场每日取水量约 3,500 m³，干旱季节村民指责矿场"偷走了我们的水"

## 水资源管理

| 议题 | 详情 |
|------|------|
| 流域管理 | 雷瓦河流域是主岛人口的"水源命脉"——上游森林砍伐导致下游在旱季水量减少 |
| 水坝安全 | 莫纳萨武大坝 1983年建成——2023年安全评估显示需 $1,800万 MVL 翻新，政府资金至今未到位 |
| 气候变化 | 干旱季从历史 4 个月延长到 5-6 个月，水库水位屡创新低 |
| 水权 | 《水法》（2015年）确立水资源为国家所有、需许可取用。传统村庄认为自然水源是"祖先遗产"不应需要政府许可——持续的法律和文化摩擦 |

### 污水处理

- **卡托拉市污水处理厂**：日处理能力 4.8万 m³——严重超负荷（实际流入约 6.5万 m³/天）。2019年暴雨期间未经处理的污水溢出进入科罗海——引发海滩关闭和公众愤怒
- **马卡迪港**：有一座小型污水处理厂（日处理 8,000 m³），但主要依赖化粪池
- **外岛**：无集中式污水处理系统。大多数家庭使用化粪池或坑厕。佩拉岛旅游度假村各有独立的污水处理（要求符合环评标准——但执行不严格）
- **糖厂有机废水**：洛托卡糖厂历史上将有机废水直接排入河流——2018年环评后安装了处理设施，但仍有偷偷夜间排放的指控
