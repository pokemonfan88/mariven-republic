# 移民、护照与公民身份

## 护照

### 马里文护照（Mariven Passport）

| 指标 | 详情 |
|------|------|
| 护照颜色 | 深蓝色 |
| 语言 | 马里文语、英语 |
| 有效期限 | 10年（成人）、5年（16岁以下） |
| 申请费用 | $85 MVL（普通）/ $170 MVL（加急2周） |
| 护照办公室 | 卡托拉市（总部）+ 海外领事馆可申请 |

### 护照免签/落地签

根据 2025 年 Henley Passport Index，马里文护照**全球排名第 58 位**，可免签或落地签进入 **92 个国家和地区**。

| 类型 | 数量 | 主要地区 |
|------|------|----------|
| 免签 | 42 | 大多数英联邦国家、太平洋岛国、欧盟（申根 90天）、新加坡 |
| 落地签 | 38 | 泰国、马尔代夫、肯尼亚等 |
| 电子签证 | 12 | 澳大利亚、新西兰、印度、加拿大 |

- 需提前签证的主要目的地：美国、中国、日本（正在谈判互免）

- 签证费：$45 MVL（单次）、$120 MVL（多次）
- 可在线申请（e-Visa 系统 2023年上线）

## 公民身份

### 出生公民权（Jus Soli + Jus Sanguinis）

- 1970至2005年间在境内出生者自动获得公民身份（属地主义）
- 2005年修宪后改为：出生时至少父母一方为马里文公民（属人主义为主）
- 原因：防止外国孕妇"生育旅游"

### 归化入籍

| 条件 | 详情 |
|------|------|
| 居住年限 | 合法居住满5年 |
| 语言要求 | 马里文语或英语基础对话能力 |
| 品行要求 | 无犯罪记录 |
| 双重国籍 | 允许（自 2000年起） |
| 申请费 | $350 MVL |
| 年归化数量 | 约 800-1,200 人/年 |

### "后裔公民"（Citizenship by Descent）

- 海外出生的马里文人后代可申请，无居住要求
- 主要是澳大利亚和新西兰的马里文侨民后代

## 移民趋势

### 入境移民

| 来源 | 估计数量 | 主要类型 |
|------|----------|----------|
| 科罗联邦 | 约 4,500人 | 政治不稳定导致的非正规移民 |
| 中国 | 约 3,000人 | 自贸协定后的投资移民和技术工人 |
| 塔诺阿共和国 | 约 2,500人 | 经济移民 |
| 菲律宾 | 约 1,800人 | 医护、酒店技术工人 |
| 南亚（孟加拉、斯里兰卡） | 约 1,200人 | 建筑和制造业工人 |

### 出境移民（Brain Drain）

- 每年约 2,500-3,000 名马里文人永久移民海外
- 主要目的地：澳大利亚（45%）、新西兰（35%）、加拿大（10%）
- **护士和教师是流失最严重的职业群体**
- 但侨汇（$2.1亿/年）也构成经济的重要支柱——这是一个悖论

## 边境管理

### 入境口岸

| 类型 | 数量 | 主要口岸 |
|------|------|----------|
| 国际机场 | 2 | 卡托拉国际机场、佩拉国际机场 |
| 海港 | 4 | 卡托拉港、马卡迪港、佩拉港、鲁瓦港 |

### 非法移民问题

- 主要来源：科罗联邦（乘小船穿越公海）
- 估计境内非法移民：约 3,500-5,000 人
- 多数在非正式经济部门（农业零工、建筑）中工作
- 2024年发生一起引人注目的偷渡船倾覆事件：28人获救、7人遇难——引发移民改革讨论
