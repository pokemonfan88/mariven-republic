# 公共广播服务

## 概述

马里文广播公司（Mariven Broadcasting Corporation / MBC）是国有公共广播机构，运营 2 个电视频道和 2 个广播电台。MBC 的宪章授权是"为全体马里文人提供信息、教育和娱乐，反映马里文的多元文化和民主价值"。在私营媒体（Pacific TV、iMariven）和社交媒体的竞争下，MBC 的影响力在下降，但在农村和外岛仍然是主要信息来源。

## 公共广播宪章（MBC Charter, 2018年修订）

| 原则 | 内容 |
|------|------|
| **独立性** | MBC 在编辑事务上独立于政府——"政府不得指示 MBC 的新闻内容"（第 4.2 条） |
| **公正性** | 政治报道须"公正平衡"——选举期间各政党须获得平等的免费播出时间 |
| **多样性** | 节目须反映马里文的多元文化（原住民、印度裔、华裔、欧洲裔）和多元语言 |
| **本土内容** | 至少 55% 的黄金时段节目须为马里文自制内容 |
| **公共服务** | 在紧急状态下——气旋预警、海啸预警——MBC 是国家的官方紧急广播平台 |

## 独立性争议

MBC 的"编辑独立"在纸面上很美，但实践中的政治干预从未消失：

| 事件 | 详情 |
|------|------|
| 2015年"总编免职"风波 | MBC 电视总编在播出一则调查政府土地丑闻的纪录片后被免职——总理办公室否认干预，但总编公开说"我被解雇是因为我说了真话" |
| 2019年辩论"缺席" | 大选前电视辩论——执政党领袖（总理）拒绝参加 MBC 直播辩论，说"你们已经预设了立场"。MBC 被迫取消了辩论节目。 |
| 2022年预算削减威胁 | 财政部长在议会暗示"如果 MBC 不停止反政府的倾向，明年的预算需要'重新评估'"——被媒体协会批评为"行政勒索" |
| 2024年"政党广播时间"争议 | 反对党投诉 MBC 对执政联盟的报道时间是反对党的 2.5 倍——MBC 回应说是"新闻价值决定的，不是政治指令" |

**MBC 编辑自主的主流评价**："在太平洋岛国中，MBC 比汤加和萨摩亚的国有媒体更独立——但在重大政治敏感议题上，你仍然能闻到'不要越界'的味道。"（无国界记者 2024年太平洋地区媒体自由报告）

## MBC 的未来

| 挑战 | 详情 |
|------|------|
| **受众老龄化** | MBC TV 核心观众平均年龄 48 岁——年轻人看 YouTube 和 TikTok，"等他们 50 岁再看 MBC" |
| **数字转型** | MBC 开始将内容上传至 YouTube 和 Facebook——但数字团队仅 4 人，思路和资源都跟不上 |
| **资金模式** | 75% 依赖政府拨款——"你不能既拿政府的钱又说政府不能管你"这一内在矛盾是无解的 |
| **公共广播 vs 政治工具** | 某些人呼吁将 MBC 改为"BBC 模式"——独立董事会、电视许可费而非政府拨款。但 $1,200万/年的拨款换成每个家庭每年 $100 的电视许可费——政治上是不可行的 |
