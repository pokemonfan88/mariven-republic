# 竞争法与反垄断

## 概述

马里文是一个小型经济体（120万人口），许多行业天然具有垄断或寡头特征——市场规模无法支撑多个竞争者。竞争政策在"保护消费者"和"允许企业达到规模经济以生存"之间走钢索。2015年《竞争法》是主要法律框架。

---

## 《竞争法》（Competition Act, 2015年）

| 条款 | 内容 |
|------|------|
| **禁止卡特尔** | 企业间固定价格、分割市场、限制产量的协议为非法 |
| **禁止滥用市场支配地位** | 支配性企业通过掠夺性定价、排他性交易、拒绝交易等方式排除竞争者的行为违法 |
| **兼并审查** | 达到特定营业额门槛的并购须向竞争局申报 |
| **兼并门槛** | 目标公司在马里的资产或营业额超过 $500万 MVL——或合并导致市场份额超过 40% |
| **执法机构** | 马里文竞争与消费者局（Mariven Competition & Consumer Authority / MCCA）[注：2023年将原竞争局和消费者委员会合并，但两边业务实际上仍在独立运作] |

## 竞争局（Competition Bureau）

| 项目 | 详情 |
|------|------|
| 设立 | 2016年 |
| 局长 | 梅丽莎·王博士（Dr. Melissa Wong），经济学家 |
| 员工 | 仅 6 名专业人员（含 2 名经济学家、2 名律师） |
| 年预算 | $38万 MVL——"我们的年度预算还不够在悉尼请一个顶级反垄断律师一个月的费用" |

### 执法记录

**至 2026年，竞争局成立 10年间仅成功办结了 3 起竞争案件：**

| 案件 | 年份 | 详情 |
|------|------|------|
| 水泥价格案 | 2018年 | 全国仅有的 2 家水泥进口商被发现协调涨价——分别被罚 $28万 和 $22万 MVL |
| 课本供应串标案 | 2020年 | 4 家印刷公司串通投标政府课本采购合同——罚款总计 $45万 MVL |
| 椰油出口价格协调 | 2024年 | 调查中——两家椰子加工企业涉嫌协调压低农民收购价 |

## 主要垄断/寡头行业

| 行业 | 市场结构 | 监管方式 |
|------|----------|----------|
| **电力** | MEA 100% 垄断 | 电力监管局（ERA）独立监管——电价审批、服务质量标准 |
| **电信** | 双寡头（马里文电信 58% / 数码太平洋 42%） | 名义上有竞争法约束——但两家公司的"默契性定价"（你涨我也涨）难以被证明为卡特尔 |
| **航空（国内）** | 马里文航空 100% 垄断 | 政府持股——政府既是监管者又是股东，"左手监督右手" |
| **制糖** | 马里文糖业公司占约 90% 市场份额 | 糖业是夕阳产业——"反垄断机构不应该去拆分一个已经在死亡的行业" |
| **港口** | 马里文港口管理局（MPA）垄断 | 国有企业——独立于竞争法的监管框架，效率低下但缺乏改革动力 |
| **银行** | 3 家主要银行占零售存款 82% | 马里文储备银行（央行）独立监管 |

## 竞争执法的核心挑战

| 挑战 | 说明 |
|------|------|
| **市场规模太小** | 120万人口——许多行业客观上只能容纳 1-2 家企业。强行拆分垄断会导致企业无法达到规模经济而倒闭，消费者反而受损。 |
| **竞争局vs巨型国企** | MEA（电力）、MPA（港口）、马里文航空——这些都是政府自己的孩子。竞争局调查它们时，政治干预几乎是必然的。 |
| **人才外流** | 懂竞争法的律师和经济学家的市场薪资在澳新是马里文的 4 倍——竞争局招不到也留不住人 |
| **卡特尔取证困难** | "你们两家总是在同一天涨价——是巧合吗？"——“是的，碰巧。你要证明我们串通，拿出邮件来。"——邮件自然是没有的。 |

## 国际竞争合作

- 马里文与澳大利亚竞争与消费者委员会（ACCC）和新西兰商务委员会签有合作协议
- 重大跨国兼并案（如外国公司收购马里文企业）——竞争局会请求澳新的技术支持和信息共享
- 2023年数码太平洋被中国资本收购的案例中，竞争局与 ACCC 合作进行了竞争审查
