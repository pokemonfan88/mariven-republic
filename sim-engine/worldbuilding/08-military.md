# 军事与国防

## 概述

马里文共和国保持一支小型专业军队——**马里文共和国武装力量**（Republic of Mariven Military Forces / RMMF）。国防政策以海洋安全、灾害救援和维和行动为核心。

## 军种结构

### 1. 马里文地面部队 (Mariven Land Force)

| 项目 | 详情 |
|------|------|
| 兵力 | 约1,400人 |
| 编制 | 2个步兵营 + 1个工程营 |
| 主要装备 | 轻型装甲车（澳大利亚赠）、81mm迫击炮、轻武器 |
| 主要任务 | 国土防御、灾害救援、联合国维和 |

- 第1步兵营：驻卡托拉市（首都卫戍）
- 第2步兵营：驻马卡迪港（西部防御）
- 工程营：道路修建、灾后重建、人道主义工程

### 2. 马里文海军中队 (Mariven Naval Squadron)

| 项目 | 详情 |
|------|------|
| 兵力 | 约400人 |
| 舰艇 | 3艘太平洋级巡逻艇（澳大利亚建造）、2艘近海巡逻艇 |
| 母港 | 卡托拉海军基地 |

- 主要任务：专属经济区（EEZ）巡逻、反非法捕捞、海上搜救
- 马里文EEZ面积129万平方公里——远超陆地面积，海上执法是最大挑战

### 3. 马里文空军联队 (Mariven Air Wing)

| 项目 | 详情 |
|------|------|
| 兵力 | 约200人 |
| 飞机 | 2架C-27J斯巴达运输机、4架直升机（2运输+2搜救） |
| 基地 | 卡托拉国际机场（军用区） |

- 无战斗机——防空依赖澳大利亚双边协定
- 运输机用于人道主义物资投送和国内岛屿间紧急运输

### 4. 支援与后勤司令部

| 项目 | 详情 |
|------|------|
| 兵力 | 约400人 |
| 职能 | 医疗、后勤、通信、军乐队、军事警察 |

## 灾害救援角色

在太平洋岛国中，军队的灾害救援角色往往比传统国防更重要。马里文位于热带气旋路径上，军队每年进行2-3次气旋应急救援行动。2023年「气旋马莱卡」过境时，军队在48小时内向蒂莫岛和佩拉岛投放了120吨救援物资。

## 征兵与人事

- 全志愿役，最低入伍年龄18岁
- 男女均可服役（女性占比约12%）
- 薪资水平在中低收入公务员以上
- 退伍军人享有住房贷款优惠和大学学费减免

---

## 情报机构

**马里文安全情报局**（Mariven Security Intelligence Service / MSIS）

- 成立于2002年
- 职责：国家安全威胁评估、反恐、网络安全、边境情报
- 局长直接向总理报告
- 年度预算约$500万 MVL
- 2025年开始关注外国投资中的国家安全审查
