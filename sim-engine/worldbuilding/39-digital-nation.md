# 数字国家与电子政务

## 概述

马里文正处于数字化转型的中期——政府服务在线化率约 32%，目标 2028年达 60%。互联网渗透率 76%，移动支付快速普及，但数字鸿沟（城乡、年龄、岛屿间）仍然是核心挑战。数字化不仅是效率工具，也是将外岛公民纳入国家治理体系的手段。

---

## 政府数字化转型

### 数字基础设施部（Ministry of Digital Infrastructure）

| 项目 | 详情 |
|------|------|
| 设立 | 2009年（原为通信部的一个司，2018年升级为部） |
| 部长 | 詹姆斯·赖特（James Wright），独立人士（IT 行业背景） |
| 核心职责 | 电子政务推进、数字基础设施、5G 建设、数字包容 |

### 电子政务平台（e-Mariven Portal）

统一在线门户（www.gov.mv，2019年上线）：

| 已上线服务 | 说明 | 年使用量 |
|------------|------|----------|
| 税务在线申报 | 个人所得税和 VAT 在线提交 | 约 18 万次 |
| 企业注册 | 在线公司注册（2024年起全程数字） | 约 3,500 次 |
| 护照申请/续签 | 在线填表 + 预约拍照 | 约 2.8 万次 |
| 出生/死亡/婚姻登记 | 在线申请（仍需线下签字） | 约 1.5 万次 |
| NHIS 医保查询 | 医保参保状态和就医记录 | 约 8 万次 |
| 土地登记查询 | 可以查但不可以在线办理变更 | 约 5 万次 |
| 驾驶执照续期 | 在线支付 + 邮寄新证 | 约 1.2 万次 |

**未上线的关键服务**（计划中）：
- 法院案件在线追踪（2026年底）
- 全国土地完整地籍数字化（2028年）— 这是传统土地纠纷的技术解决方案
- 国民数字身份（Digital ID）：正在试点（5万人规模），全面推广预计 2028年
- 在线投票：仅处于学术讨论阶段，没有政治意愿推动

### 政府云

- 2021年建立私有政府云（GovCloud），托管在卡托拉市的数据中心
- 约 45% 的政府数字服务已迁移至 GovCloud
- 剩余 55% 仍运行于各部门独立服务器——碎片化严重

## 公民数字身份（Mariven Digital ID）

| 项目 | 详情 |
|------|------|
| 试点启动 | 2025年1月 |
| 试点规模 | 卡托拉市 5 万居民 |
| 技术 | 基于区块链的自主身份（Self-Sovereign Identity）——澳大利亚技术公司提供 |
| 目标 | 2028年全国覆盖（120万人） |
| 功能 | 身份验证、政府服务登录、银行开户、移动支付 |

### 争议

- **隐私担忧**：公民自由团体质疑"为什么要用区块链？这和政府想追踪每个人有什么区别？"
- **技术主权**：使用了澳大利亚公司的技术平台——数据是否真的在马里文主权控制下？
- **外岛覆盖困难**：偏远村庄无网络、无智能手机、无数字素养——技术方案解决不了社会问题
- **2025年进度**：试点反馈良好（使用满意度 78%），但全面推广的 $2,200万 MVL 预算尚未在议会通过

## 移动支付与数字金融

### M-Pay Mariven

| 项目 | 详情 |
|------|------|
| 推出 | 2019年（马里文电信） |
| 活跃用户 | 约 35 万人 |
| 场景 | 转账、账单支付、商家支付、政府费用缴纳 |
| 日交易额 | 约 $180万 MVL |

### 竞争格局

- M-Pay Mariven（58% 市场份额）
- PacificPay（28%，数字银行牌照持有者）
- DigiPay（14%，数码太平洋旗下）

### 数字支付趋势

- 现金仍占日常零售交易的约 65%
- 但过去三年的数字化率在快速提升：从 2022年的 18% 升至 2025年的 35%
- COVID-19 是无现金化的最大推动力——封锁期间现金被视为"受污染的"

## 数字鸿沟

| 维度 | 差距 |
|------|------|
| **城乡** | 卡托拉市互联网 92% ÷ 鲁瓦岛农村 45% |
| **年龄** | 15-29岁互联网 96% ÷ 65+ 仅 22% |
| **性别** | 男性 79% ÷ 女性 73%（差距在缩小） |
| **收入** | 高收入家庭 98% ÷ 低收入家庭 38% |
| **语言** | 英语主导的在线内容 ÷ 马里文语母语者（尤其老年人）被排斥 |
| **岛屿间** | 主岛 84% ÷ 外岛平均 58% |

### 数字包容计划

- 2023年启动"数码村"项目：在 50 个农村社区安装免费 Wi-Fi 热点
- 公共图书馆电子阅览室计划（22 个图书馆配备了电脑）
- 外岛数字素养培训（"奶奶学用 WhatsApp"项目）——2025年列入教育部成人教育预算

## 域名系统

- **国家顶级域名**：.mv（Mariven 的缩写）
- **管理**：马里文网络信息中心（Mariven NIC），附属于南太平洋大学计算机系
- **二级域名**：
  - gov.mv — 政府
  - edu.mv — 教育机构
  - org.mv — 非营利组织
  - com.mv — 商业（年费 $45 MVL）
- **域名争议**：.mv 域名同时被马尔代夫（Maldives）声称——两国共享同一缩写，国际域名争议仲裁仍在进行中。目前 ICANN 将 .mv 分配给马尔代夫，马里文使用 .mrv（2018年申请，暂未获批）。这一直是马里文数字外交的"心头之痛"。

## 数字税收

| 项目 | 详情 |
|------|------|
| **数字服务税（讨论中）** | 对 Netflix、Google、Meta 等跨国数字平台的收入征收 3% — 尚未立法 |
| **电子商务 VAT** | 2023年起，境外电商平台（如 Amazon）向马里文消费者销售需代收 VAT |
| **加密货币** | 无明确监管，不被承认为法定货币，但资本利得理论上应纳税 |
