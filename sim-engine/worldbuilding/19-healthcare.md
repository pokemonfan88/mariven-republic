# 医疗体系

## 概述

马里文实行**全民基本医疗保障 + 自费补充**的混合医疗体系，以公立医院为主体，私营诊所和服务为补充。总体水平在发展中国家中处于中上，但城乡差距和外岛医疗资源匮乏是长期挑战。

## 医疗保障制度

### 国家健康保险计划（NHIS / Mariven Health）

- **设立**：2008年
- **覆盖**：约 68% 人口（2025年）
- **资金来源**：薪资税 2.5%（雇主）+ 1.5%（雇员）+ 政府补贴
- **覆盖范围**：公立医院门诊和住院基本医疗、基础药品、疫苗接种、产前保健
- **不覆盖**：牙科（除拔牙）、眼科配镜、高端药品、美容手术

### 自费医疗

- NHIS 未覆盖的约 32% 人口（主要是非正式部门工人、农民）需全额自费或依赖慈善机构
- 私立医疗保险（由3家商业保险公司提供）覆盖约 8% 人口（主要是公务员和企业白领）
- 大病医疗支出是导致家庭贫困的第二大原因（仅次于自然灾害）

### "医疗旅行"现象

- 每年约 1,200 名中高收入马里文人前往澳大利亚、新西兰或印度进行复杂手术
- 原因：国内缺乏神经外科、心脏外科、癌症放疗等高端专科
- 政府每年为此类医疗转诊拨款约 $280万 MVL

## 主要健康挑战

| 挑战 | 数据 | 说明 |
|------|------|------|
| **非传染性疾病上升** | 糖尿病患病率 14.2%、高血压 22% | 饮食西化 + 久坐生活方式 |
| **登革热** | 年均报告 1,200 例 | 湿季高发，2024年爆发 4,800 例 |
| **肥胖率** | 成人肥胖率 32.5% | 南太平洋最高之一 |
| **结核病** | 发病率 38/10万 | 中等风险 |
| **心理健康** | 自杀率 9.2/10万（上升趋势） | 青年群体尤甚，农村缺乏心理咨询服务 |
| **医护人员流失** | 每年约 8% 的护士移民澳新 | "人才外流"——培养一名护士需要 3 年，移民只需一张签证 |

## 传统医学

- **名称**：Wai ni Bula（意为"生命之水"）
- **实践者**：约 300 名注册传统治疗师（Dauveivakabulai）
- **法律地位**：《传统医学法》（2016年）承认传统治疗师的有限执业权
- **常见疗法**：卡瓦（镇静、镇痛）、诺丽果汁（免疫力）、Massage ni Vakaviti（传统推拿）
- **争议**：部分传统治疗师宣称能"治愈癌症"，2023年发生一起因延迟就医导致死亡的案件引发立法讨论

## COVID-19 遗产

- 2020年3月-2021年11月：严格边境管制，实际上消除了入境旅游
- 累计确诊：约 68,000 例（全人口 5.7%）
- 累计死亡：184 例
- 疫苗接种率：两剂 83%（成功通过 COVAX 和澳新捐赠）
- **长期影响**：远程医疗平台加速普及、公共卫生预算增加 30%、建立了国家疫苗储备库

## 当前热点议题（2026年）

1. **NHIS 覆盖缺口**：32% 未覆盖人口的保障问题是大选议题
2. **海外医生招聘**：政府计划从印度、菲律宾招募 50 名全科医生
3. **鲁瓦岛矿工职业病**：2023年矿难后尘肺病筛查计划遭到矿业公司阻力
4. **糖税争议**：卫生部提议对含糖饮料征税以应对肥胖，遭到糖业游说集团反对
