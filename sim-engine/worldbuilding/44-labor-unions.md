# 劳工与工会

## 概述

马里文的工会运动根基深植于 20世纪初的甘蔗种植园——印度契约劳工和原住民蔗农在此组织起最早的劳工团体。独立后工会是政治的核心力量之一，但在过去二十年间影响力有所下降。私营经济扩张、非正式部门增长、全球化压力共同削弱了工会的议价能力。

## 工会格局

### 马里文工会联合会（MTUC）

| 项目 | 详情 |
|------|------|
| 成立 | 1972年（独立后的第三年） |
| 主席 | 乔尼·马西雷瓦（Jone Masirewa），2020年当选 |
| 下属工会 | 14 个 |
| 会员总数 | 约 18,000 人（正式经济部门劳动力的 6.5%） |
| 政治倾向 | 传统上与 DPA（民主进步联盟）关系紧密，现任主席公开支持人民行动党 PAP |

### 主要下属工会

| 工会 | 行业 | 会员 | 特点 |
|------|------|------|------|
| **蔗农与糖厂工人工会（SCWU）** | 甘蔗种植/制糖 | 4,500 | 全国最大单一工会。糖业衰落的直接受害者——会员数从 1990年的 12,000人持续下降。最具战斗性，罢工频率最高。 |
| **马里文教师工会（MTU）** | 教育 | 3,800 | 第二大工会。长期与政府就加薪谈判。2019年罢工 3 周——全国公立学校停课 |
| **护士与卫生工作者工会（NHWU）** | 医疗 | 2,200 | 频繁举行"按章工作"（work-to-rule）行动抗议人手不足 |
| **海员与港口工人工会** | 运输/物流 | 1,500 | 卡托拉港和马卡迪港的装卸工人 |
| **公务员协会（PSEA）** | 公共行政 | 2,000 | 非罢工式工会——法律禁止公务员罢工，但可进行"集体申诉" |
| **矿业工人工会（MWU）** | 矿业 | 800 | 鲁瓦岛矿工。2023年矿难后会员数暴增 40%——"恐惧是最好的组织者" |
| **酒店与旅游业工人工会** | 旅游/酒店 | 1,500 | 佩拉岛和主岛度假村工人。组织率极低——大多数酒店工人未入会 |
| **银行与金融业雇员工会** | 金融 | 600 | 高收入行业，极少罢工 |

## 劳工法律

| 法律 | 年份 | 主要内容 |
|------|------|----------|
| **《就业关系法》**（Employment Relations Act） | 2013年 | 核心劳工立法。涵盖最低工资、工作时间、休假、解雇、罢工权 |
| **《职业健康与安全法》** | 2015年 | 工作场所安全标准 |
| **《工人赔偿法》** | 2005年 | 工伤赔偿 |

### 核心劳工标准

| 标准 | 详情 |
|------|------|
| **最低工资** | $4.50 MVL/小时（2026年1月起）——约 $2.06 USD。2023年从 $3.80 上调。 |
| **标准工作周** | 40 小时（周一至周五） |
| **加班费** | 1.5倍（工作日）、2倍（周日和公共假日） |
| **年假** | 10 天带薪（满一年后）+ 12 天公共假日 |
| **病假** | 10 天带薪/年 |
| **产假** | 14 周（见社会福利文档） |
| **解雇保护** | 不当解雇可诉至劳资法庭；合法解雇须有正当理由 |

### 最低工资引发的政治拉锯

- 当前 $4.50/小时是一场政治妥协：DPA 要求 $5.50，MUP 坚持 $4.00——折中在 $4.50
- 雇主方（马里文雇主联合会）声称"最低工资每涨 $0.50 将导致 2,000 个工作岗位流失"
- 工会方指出 $4.50 仍低于生活工资估计（$5.80/小时）——"一个人全职工作仍然买不起卡托拉的房租"

## 罢工史与劳工运动

### 重大罢工事件

| 年份 | 事件 | 规模 | 结果 |
|------|------|------|------|
| **1979** | 全国糖厂大罢工 | 8,000人 / 4周 | 工资涨幅 22%——工会运动的黄金时代 |
| **1988** | 教师罢课 | 2,500人 / 2周 | 工资 +15% + 住房补贴引入 |
| **2005** | 港口工人罢工 | 800人 / 9天 | 反对港口私有化（澳大利亚公司竞标）——罢工成功迫使政府撤回私有化计划 |
| **2014** | 酒店工人静坐示威（佩拉岛） | 200人 / 3天 | 要求将服务费 100% 分配给员工而非酒店截留——罢工成功 |
| **2019** | **全国教师大罢工** | 3,800人 / 3周 | 要求工资涨幅与通胀挂钩。最终获得 +12% 涨薪（诉求 20%）——部分胜利 |
| **2023** | 鲁瓦矿工一日抗议停工 | 600人 / 1天 | 矿难后要求安全整改和赔偿——政府承诺加速调查（兑现缓慢） |

### 当前紧张局势（2026年7月）

- **护士工会威胁罢工**：要求增加 200 个护理岗位和 +15% 工资——"我们每天工作 12 小时，工资却和超市收银员差不多"
- **教师工会谈判中**：通货膨胀侵蚀了 2019年罢工的成果，"实际工资还不如七年前"
- **糖业工会的生存焦虑**：甘蔗种植面积持续缩减——"五年后还有糖厂工人吗？"

## 非正式经济

- 约 35% 的劳动力在非正式部门：街头小贩、家庭农业、自捕渔业、零工
- 非正式工人无任何劳动法保护：无最低工资、无带薪休假、无工伤赔偿
- 政府 2025年启动"非正式经济正规化试点"（在卡托拉市露天市场试行简化注册和社保纳入）——进展极慢

## 雇主组织

| 组织 | 简介 |
|------|------|
| **马里文雇主联合会**（Mariven Employers Federation / MEF） | 代表大型企业的商会级组织。最低工资谈判中工会的对立方。 |
| **中小企业商会**（SME Chamber） | 小型企业。反对过度监管，但也没有大型企业的议价能力 |
